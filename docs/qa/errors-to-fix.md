# سجل الأخطاء — للإصلاح لاحقاً

> **الهدف:** تسجيل كل خطأ يذكره المستخدم، مع نوعه وتفاصيله
> **خطة العمل:** بعد الانتهاء من التسجيل → نبدأ الإصلاح

---

## قائمة الانتظار

---

### 1. 🟡 التمرين 1 (A1 Übungsbuch Lektion 01) — "اختر" مضللة

| الحقل | القيمة |
|-------|--------|
| **النوع** | 🟡 متوسط — خطأ في التعليمات |
| **النسخة العربية** | `a1/ubungsbuch/lektion-01/` |
| **النسخة الإنجليزية** | `a1/ubungsbuch/lektion-01.en/` |
| **المسمى** | التمرين 1: املأ الفراغ 🖊️ |
| **التعليمات الحالية** | "اختر التحية المناسبة للوقت" |
| **data-type** | `text` (يكتب المستخدم الإجابة) |
| **المشكلة** | كلمة **"اختر"** توحي بأن المستخدم سيختار من خيارات (choice/radio)، لكن التمرين من نوع text يعني على المستخدم **كتابة** الإجابة بنفسه |

**الإصلاح المقترح:** استبدال "اختر" بـ "اكتب" → "**اكتب** التحية المناسبة للوقت"

> ⚠️ ينطبق على النسختين: العربية والإنجليزية

### 2. 🟡 التمرين 2 (A1 Übungsbuch Lektion 01) — text بدل match

| الحقل | القيمة |
|-------|--------|
| **النوع** | 🟡 متوسط — تحسين نوع التمرين |
| **النسخة العربية** | `a1/ubungsbuch/lektion-01/` |
| **النسخة الإنجليزية** | `a1/ubungsbuch/lektion-01.en/` |
| **المسمى** | التمرين 2: صِل (Match) 🔗 |
| **الوصف** | صِل التحية الألمانية بالمعنى العربي بكتابة الحرف المناسب |
| **الوضع الحالي** | `data-type="text"` — حقل إدخال نصي يكتب فيه المستخدم الحرف (أ, ب, ج...) |
| **data-answers** | `['د', 'أ', 'ه', 'ب', 'ج', 'ز', 'و']` (7 إجابات ← 7 أسئلة ✓) |
| **المطلوب** | تحويله إلى `data-type="match"` مع `data-options` لتظهر قائمة منسدلة |

**الإصلاح المقترح:**
- تغيير `data-type` من `"text"` إلى `"match"`
- إضافة `data-options='["أ","ب","ج","د","ه","و","ز"]'`

> ⚠️ ينطبق على النسختين: العربية والإنجليزية

### 3. 🟡 التمرين 3 (A1 Übungsbuch Lektion 02) — "صِل" لكنه text

| الحقل | القيمة |
|-------|--------|
| **النوع** | 🟡 متوسط — تحسين نوع التمرين |
| **النسخة العربية** | `a1/ubungsbuch/lektion-02/` |
| **النسخة الإنجليزية** | `a1/ubungsbuch/lektion-02.en/` |
| **المسمى** | التمرين 3: صِل المهنة بالعربية 🔗 |
| **التعليمات** | "صِل المهنة الألمانية بالمعنى العربي" |
| **الوضع الحالي** | لا يوجد `data-type` ← افتراضياً `text` — حقل إدخال نصي |
| **data-answers (عربي)** | `["ب","أ","ه","ز","ج","د","ح","و","ي","ط"]` (أحرف عربية) |
| **data-answers (إنجليزي)** | `["B","A","E","G","C","D","H","F","J","I"]` (أحرف لاتينية) |
| **الجدول** | عمودان فقط (الألمانية \| العربية / German \| English) — لا يوجد عمود للإجابة |
| **المشكلة** | التمرين مكتوب عليه "صِل" (Match) لكنه من نوع text، والجدول لا يحتوي عموداً للإجابة |

**الإصلاح المقترح:**
- إضافة `data-type="match"` لكلتا النسختين
- إضافة `data-options` عربية: `'["أ","ب","ج","د","ه","و","ز","ح","ط","ي"]'`
- إضافة `data-options` إنجليزية: `'["A","B","C","D","E","F","G","H","I","J"]'`
- إعادة تصميم الجدول ليشمل عمود # وعمود للإجابة (مثل باقي تمارين match)

> ⚠️ ينطبق على النسختين: العربية والإنجليزية

### 4. 🟡 التمرين 4 (A1 Übungsbuch Lektion 02) — ✅ يكشف الإجابة مسبقاً

| الحقل | القيمة |
|-------|--------|
| **النوع** | 🟠 عالي — الإجابة مكشوفة قبل الحل |
| **النسخة العربية** | `a1/ubungsbuch/lektion-02/` |
| **النسخة الإنجليزية** | `a1/ubungsbuch/lektion-02.en/` |
| **المسمى** | التمرين 4: اختر — du ولا Sie؟ 🎯 |
| **data-type** | `choice` |
| **data-answers** | `["A","B","A","B","B","A","B","A"]` (صحيحة) |
| **المشكلة** | جدول التمرين يحتوي ✅ في الخانة الصحيحة و ⬜ في الخانة الخاطئة — هذا **يكشف الإجابة** للطالب قبل أن يحل السؤال بنفسه. الطالب يقرأ الجدول فقط ويختار دون تفكير |

**الإصلاح المقترح:**
- استبدال ✅ و ⬜ في الجدول بـ ⬜ في كلا الخانتين (du و Sie) — أي جميع الخانات ⬜ ⬜
- عندها فقط يضطر الطالب للتفكير قبل اختيار الإجابة

> ⚠️ ينطبق على النسختين: العربية والإنجليزية

### 5. 🟡 التمرين 9 (A1 Übungsbuch Lektion 02) — ? تظهر في البداية (Bidi)

| الحقل | القيمة |
|-------|--------|
| **النوع** | 🟡 متوسط — Bidi rendering |
| **النسخة العربية** | `a1/ubungsbuch/lektion-02/` |
| **النسخة الإنجليزية** | غير موجود مقابل إنجليزي مباشر |
| **المسمى** | التمرين 9: اختر الإجابة الصحيحة — المهنة والمكان 🎯 |
| **data-type** | `choice` |
| **data-answers** | `["B","A","C","B","C","C"]` |
| **المشكلة** | الجمل الألمانية تنتهي بـ `?` (مثل `Wo arbeitet ein Arzt?`) لكن بسبب RTL تظهر علامة الاستفهام في **البداية**: `?Wo arbeitet ein Arzt` |

**الإصلاح المقترح:**
- إضافة `&lrm;` (Left-to-Right Mark) بعد كل `?` في الجدول
- مثال: `Wo arbeitet ein Arzt?&lrm;`

> ⚠️ ينطبق على النسخة العربية فقط (الإنجليزية LTR فلا مشكلة)
