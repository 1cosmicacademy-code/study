# The Accusative Case (Akkusativ) in German

🎯 The **Accusative case** is the second case in German. We use it mainly for the direct object — the thing or person that receives the action.

---

## 💡 When do we use the Accusative?

### 1. Direct Object

It answers **"wen?"** (whom?) or **"was?"** (what?)

```
Ich sehe den Lehrer. → Wen sehe ich? → Den Lehrer.
I see the teacher.
```

```
Er isst einen Apfel. → Was isst er? → Einen Apfel.
He is eating an apple.
```

### 2. After Certain Verbs

Many verbs require the Accusative without a preposition:

| Verb (Akkusativ) | Meaning |
|-----------------|---------|
| **haben** | to have |
| **essen** | to eat |
| **trinken** | to drink |
| **sehen** | to see |
| **hören** | to hear |
| **kaufen** | to buy |
| **lieben** | to love |
| **kennen** | to know (a person) |
| **lesen** | to read |
| **schreiben** | to write |
| **brauchen** | to need |
| **nehmen** | to take |

### 3. After Certain Accusative Prepositions

| Preposition | Meaning |
|------------|---------|
| **durch** | through |
| **für** | for |
| **gegen** | against |
| **ohne** | without |
| **um** | around |
| **bis** | until |
| **entlang** | along |

**Examples:**
- `Ich gehe **durch den** Park.` — I walk through the park.
- `Das Geschenk ist **für dich**.` — The gift is for you.
- `**Ohne dich** kann ich nicht leben.` — I can't live without you.

---

## ⚠️ Article Changes in the Accusative

| Case | der (masculine) | die (feminine) | das (neuter) | die (plural) |
|------|----------------|---------------|-------------|-------------|
| **Nominativ** | der | die | das | die |
| **Akkusativ** | **den** | die | das | die |

✅ **Only masculine (der) changes:** der → den  
The rest (die, das, die) stay the same.

### Complete Article Table

| Article Type | Masculine (m) | Feminine (f) | Neuter (n) | Plural (pl) |
|-------------|--------------|-------------|-----------|-------------|
| **der →** | **den** | die | das | die |
| **ein →** | **einen** | eine | ein | --- |
| **kein →** | **keinen** | keine | kein | **keine** |
| **mein →** | **meinen** | meine | mein | **meine** |

---

## 🎯 Golden Rule

```
Accusative = the direct object
Only the masculine changes: der → den, ein → einen
```

## 📝 Exercises

### Exercise 1: Complete with the correct article
1. Ich habe ___ Hund. (ein)
2. Sie kauft ___ Blumen. (die)
3. Wir sehen ___ Film. (der)
4. Er trinkt ___ Wasser. (das)
5. Ich brauche ___ Hilfe. (die)

### Exercise 2: Complete with für, durch, ohne, um, gegen
1. Das ist ___ dich. (for you)
2. Wir gehen ___ den Park. (through)
3. ___ dich gehe ich nicht. (without)
4. Wir laufen ___ den See. (around)
5. Das Medikament ist ___ Kopfschmerzen. (against)

### Exercise 3: Identify the Accusative object
1. Ich lese ein Buch.
2. Der Vater kauft einen Ball.
3. Wir hören die Musik.
4. Das Kind trinkt Milch.
5. Ich kenne diesen Mann nicht.

### Exercise 4: Change from Nominativ to Akkusativ
1. Der Tisch ist groß. → Ich sehe ___ Tisch.
2. Ein Hund bellt. → Ich höre ___ Hund.
3. Die Blume ist schön. → Ich kaufe ___ Blume.
4. Das Kind spielt. → Ich sehe ___ Kind.
5. Kein Student ist da. → Ich kenne ___ Studenten.

### Exercise 5: Choose the correct option
1. Ich habe (ein / einen / eine) Bruder.
2. Sie liebt (der / den / die) Mann.
3. Wir besuchen (das / den / die) Oma.
4. Er kauft (ein / eine / einen) Auto.
5. Ich möchte (eine / einen / ein) Kaffee, bitte.

---

## ✅ Solutions

### Solution Exercise 1
1. einen | 2. die | 3. den | 4. das | 5. die

### Solution Exercise 2
1. für | 2. durch | 3. Ohne | 4. um | 5. gegen

### Solution Exercise 3
1. ein Buch | 2. einen Ball | 3. die Musik | 4. Milch | 5. diesen Mann

### Solution Exercise 4
1. den | 2. einen | 3. die | 4. das | 5. keinen

### Solution Exercise 5
1. einen | 2. den | 3. die | 4. ein | 5. einen

---

💡 **Tip:** The easiest way to remember the Accusative — ask yourself "what?" or "whom?" after the verb. The answer is the Accusative object.
