## 🎯 Lesson Goals

- Understand how German forms compound nouns and why they are useful
- Learn the basic rules for building compounds (first part + second part)
- Recognize common patterns (noun + noun, adjective + noun, verb stem + noun, etc.)
- Avoid typical mistakes like incorrect linking elements

---

## 📚 Core Rules

1. **Basic Structure**: A compound noun joins two (or more) words together without a space. The first part modifies the second, giving it a more specific meaning. Example: *Wohnzimmer* = *Wohn* (living) + *Zimmer* (room) → "living room".
2. **Common Types**:
   - **Noun + Noun** (e.g., *Hausaufgabe* – house + task → homework)
   - **Adjective + Noun** (e.g., *schwarz‑Weiß* – black‑white → black‑white)
   - **Verb Stem + Noun** (e.g., *Waschmaschine* – wash + machine → washing machine)
   - **Preposition/Prefix + Noun** (e.g., *Nachmittag* – after + noon → afternoon)
3. **Linking Elements**: Sometimes a small element is added between parts, such as **-er**, **-ung**, **-heit**, **-schaft** to help pronunciation and indicate the relationship.

> ⚠️ **Common Error:** Writing a space between the parts (e.g., "Haus Aufgabe") is wrong. The correct form is a single word without spaces.

---

## ✅ Examples

- *das Wohnzimmer* — *Wohn* (living) + *Zimmer* (room) → living room
- *die Sonnenblume* — *Sonne* (sun) + *Blume* (flower) → sunflower
- *der Staubsauger* — *Staub* (dust) + *Sauger* (sucker) → vacuum cleaner
- *die Kindergärten* — *Kinder* (children) + *Gärten* (gardens) → kindergartens
