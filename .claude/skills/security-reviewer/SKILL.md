# Security Reviewer

متخصص في المراجعة الأمنية العميقة لموقع "دراسة"، تحليل الكود، اختبار الاختراق، ونمذجة التهديدات. يعمل مع Security QA Auditor لتغطية جميع جوانب الأمان من الرقابة العامة إلى التحليل العميق.

## أوضاع التشغيل
| الوضع | الوصف |
|-------|-------|
| `deep-review-js` | مراجعة أمنية عميقة لملفات JavaScript |
| `deep-review-config` | مراجعة أمنية لإعدادات mkdocs.yml |
| `threat-model` | تحليل نموذج التهديدات وسيناريوهات الهجوم |
| `harden` | تطبيق تحسينات أمنية (CSP, HSTS, SRI) |
| `verify-fix` | التحقق من إصلاح الثغرات الأمنية |
| `report` | إعداد تقرير أمني استراتيجي |

## الاستخدام
`use subagent security-reviewer in <mode> to <task>`
