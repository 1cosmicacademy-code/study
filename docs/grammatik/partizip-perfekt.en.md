## 💡 What is Partizip 2?

**Partizip 2** (or Partizip Perfekt) is the third form of German verbs. It is used to form:
- **Perfekt** tense: Ich habe **gekauft**.
- **Plusquamperfekt** tense: Ich hatte **gekauft**.
- **Passiv** (passive voice): Es wird **gekauft**.
- As an adjective: das **gekaufte** Auto.

---

## 📊 How to Form Partizip 2

### 1. Regular Verbs (schwache Verben)

`ge-` + verb stem + `-t`

| Infinitive | Stem | Partizip 2 |
|:----------:|:----:|:----------:|
| machen (to do) | mach | **gemacht** |
| sagen (to say) | sag | **gesagt** |
| kaufen (to buy) | kauf | **gekauft** |
| spielen (to play) | spiel | **gespielt** |
| wohnen (to live) | wohn | **gewohnt** |
| lernen (to learn) | lern | **gelernt** |

### 2. Irregular Verbs (starke Verben)

`ge-` + (changed) stem + `-en`

| Infinitive | Partizip 2 |
|:----------:|:----------:|
| sehen (to see) | **gesehen** |
| kommen (to come) | **gekommen** |
| gehen (to go) | **gegangen** |
| finden (to find) | **gefunden** |
| sprechen (to speak) | **gesprochen** |
| schreiben (to write) | **geschrieben** |
| essen (to eat) | **gegessen** |
| trinken (to drink) | **getrunken** |
| nehmen (to take) | **genommen** |
| fahren (to drive) | **gefahren** |

### 3. Verbs Ending in -ieren

**Without** ge-, just stem + `-t`

| Infinitive | Partizip 2 |
|:----------:|:----------:|
| studieren (to study) | **studiert** |
| probieren (to try) | **probiert** |
| telefonieren (to phone) | **telefoniert** |
| korrigieren (to correct) | **korrigiert** |
| organisieren (to organize) | **organisiert** |
| reparieren (to repair) | **repariert** |

### 4. Separable Verbs (trennbare Verben)

`ge-` goes between prefix and verb

| Infinitive | Partizip 2 |
|:----------:|:----------:|
| einkaufen (to shop) | **eingekauft** |
| aufstehen (to get up) | **aufgestanden** |
| anrufen (to call) | **angerufen** |
| mitbringen (to bring along) | **mitgebracht** |
| fernsehen (to watch TV) | **ferngesehen** |
| zumachen (to close) | **zugemacht** |

### 5. Inseparable Verbs (untrennbare Verben)

**Without** ge-, just prefix + stem

| Infinitive | Partizip 2 |
|:----------:|:----------:|
| verstehen (to understand) | **verstanden** |
| besuchen (to visit) | **besucht** |
| bezahlen (to pay) | **bezahlt** |
| verkaufen (to sell) | **verkauft** |
| erklären (to explain) | **erklärt** |
| bekommen (to receive) | **bekommen** |

---

## 📊 Classification Table

| Type | Rule | Example |
|:-----|:-----|:--------|
| Regular (schwach) | ge- + stem + -t | gemacht, gesagt, gekauft |
| Irregular (stark) | ge- + (changed) stem + -en | gesehen, gekommen, geschrieben |
| Mixed (gemischt) | ge- + (changed) stem + -t | gebracht, gewusst, gedacht |
| -ieren | stem + -t (no ge-) | studiert, probiert |
| Separable (trennbar) | prefix + ge- + stem + ... | eingekauft, angerufen |
| Inseparable (untrennbar) | prefix + stem + ... | verstanden, bezahlt |

---

## ✅ Examples in Sentences

- Ich habe gestern **eingekauft**. — I shopped yesterday.
- Er ist nach Hause **gekommen**. — He came home.
- Wir haben einen Film **gesehen**. — We watched a film.
- Hast du **verstanden**? — Did you understand?
- Sie hat Deutsch **studiert**. — She studied German.
- Ich bin früh **aufgestanden**. — I got up early.
