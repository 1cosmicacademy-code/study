## 1. 🤧 كلمات جديدة — Krank sein (أن تكون مريضاً)

هذي أهم الكلمات عشان توصف إحساسك لما تكون مريض:

| Deutsch | Aussprache | Arabisch |
|---------|-----------|----------|
| **krank** | كرانك | مريض |
| **gesund** | غيزوند | بصحة جيدة / سليم |
| **der Arzt / die Ärztin** | دير آرتست / دي آرتستين | دكتور / دكتورة |
| **die Praxis** | دي براكسيس | عيادة |
| **das Krankenhaus** | داس كرانكنهاوس | مستشفى |
| **die Krankmeldung** | دي كرانكملدونغ | الإبلاغ عن المرض |
| **die Krankschreibung** | دي كرانكشرايبنغ | الإجازة المرضية (شهادة طبية) |
| **der Termin** | دير تيرمين | موعد |
| **die Sprechstunde** | دي شبريششتوندي | ساعات العمل (في العيادة) |
| **das Medikament** | داس ميديكامنت | دواء |
| **die Tablette** | دي تابليته | حبة دواء / قرص |
| **die Apotheke** | دي أبوتيكه | صيدلية |
| **das Fieber** | داس فيبر | حمى / حرارة |
| **der Husten** | دير هوستن | سعال / كحة |
| **der Schnupfen** | دير شنوبفن | زكام / رشح |
| **die Kopfschmerzen** | دي كوبفشمرتسن | صداع (ألم الرأس) |
| **die Halsschmerzen** | دي هالسشمرتسن | ألم الحلق |
| **die Rückenschmerzen** | دي روكينشمرتسن | ألم الظهر |
| **die Bauchschmerzen** | دي باوخشمرتسن | ألم البطن |

> 🏷️ **تذكير:** der = مذكر، die = مؤنث (وللجمع)، das = محايد. كلمات الأمراض غالباً جمع (مثل Kopfschmerzen).

## 2. 🤒 الأعراض والجسم — Symptome und Körper

### أجزاء الجسم المهمة:

| Deutsch | Aussprache | Arabisch |
|---------|-----------|----------|
| **der Kopf** | دير كوبف | رأس |
| **der Hals** | دير هالس | حلق / رقبة |
| **der Rücken** | دير روكين | ظهر |
| **der Bauch** | دير باوخ | بطن |
| **das Auge (die Augen)** | داس آوغه (دي آوغن) | عين / عيون |
| **die Nase** | دي نازه | أنف |
| **der Mund** | دير موند | فم |
| **das Bein** | داس باين | رجل / ساق |
| **der Arm** | دير آرم | ذراع |
| **die Hand** | دي هاند | يد |

### 🔑 كيف تقول "عندي..."؟

في الألمانية، نقول **Ich habe + Symptom** (عندي + العرض):

| Deutsch | العربية |
|---------|---------|
| Ich habe **Fieber**. | عندي **حمى**. |
| Ich habe **Kopfschmerzen**. | عندي **صداع**. |
| Ich habe **Halsschmerzen**. | عندي **ألم في الحلق**. |
| Ich habe **Husten**. | عندي **سعال (كحة)**. |
| Ich habe **Schnupfen**. | عندي **زكام**. |
| Ich habe **Bauchschmerzen**. | عندي **ألم في البطن**. |
| Ich habe **Rückenschmerzen**. | عندي **ألم في الظهر**. |
| Ich habe **keine Lust**. | **ليس لدي** رغبة / نفس. |

### 🔑 كيف تقول "يؤلمني..."؟

نستخدم **weh tun** (يؤلم) + Dativ:

| Deutsch | العربية |
|---------|---------|
| **Mir tut** der Kopf **weh**. | رأسي **يؤلمني**. |
| **Mir tut** der Hals **weh**. | حلقي **يؤلمني**. |
| **Mir tut** der Bauch **weh**. | بطني **يؤلمني**. |
| **Mir tun** die Augen **weh**. | عيوني **تؤلمني**. |
| **Mir tun** die Beine **weh**. | رجلي **تؤلمني**. |

> ⚠️ **القاعدة البسيطة:**
> - **Ich habe + Symptom** = عندي... (Ich habe Kopfschmerzen.)
> - **Mir tut + Teil + weh** = ...يؤلمني (Mir tut der Kopf weh.)
> - **Mir tun + Teile (جمع) + weh** = ...يؤلمني (جمع) (Mir tun die Augen weh.)

## 3. 🗣️ محادثة — عند الطبيب (Beim Arzt)

### حوار 1: أنا مريض

**Person A:** Hallo! Wie geht es dir?
**Person B:** Mir geht es nicht gut. Ich bin krank.
**Person A:** Was hast du?
**Person B:** Ich habe Kopfschmerzen und Fieber.
**Person A:** Oh, das ist nicht schön. Gehst du zum Arzt?
**Person B:** Ja, ich habe einen Termin um 10 Uhr.

**الترجمة:**
**الشخص أ:** مرحبا! كيف حالك؟
**الشخص ب:** لست بخير. أنا مريض.
**الشخص أ:** ماذا بك؟
**الشخص ب:** عندي صداع وحرارة.
**الشخص أ:** أوه، هذا ليس جميلاً. هل تذهب إلى الطبيب؟
**الشخص ب:** نعم، عندي موعد الساعة 10.

---

### حوار 2: في العيادة

**Arzt:** Guten Tag! Was fehlt Ihnen?
**Patient:** Guten Tag, Herr Doktor. Mir geht es nicht gut.
**Arzt:** Haben Sie Schmerzen?
**Patient:** Ja, mir tut der Kopf weh und ich habe Husten.
**Arzt:** Haben Sie Fieber?
**Patient:** Ja, ich habe 38 Grad Fieber.
**Arzt:** Okay. Ich schreibe Sie krank. Hier ist die Krankschreibung für drei Tage.
**Patient:** Danke schön!

**الترجمة:**
**الطبيب:** يوم سعيد! ما بك؟
**المريض:** يوم سعيد يا دكتور. لست بخير.
**الطبيب:** هل عندك ألم؟
**المريض:** نعم، رأسي يؤلمني وعندي كحة.
**الطبيب:** هل عندك حرارة؟
**المريض:** نعم، حرارتي 38 درجة.
**الطبيب:** حسناً. سأكتب لك إجازة مرضية. هذه شهادة الإجازة المرضية لثلاثة أيام.
**المريض:** شكراً جزيلاً!

---

### حوار 3: اتصل بالعمل

**Person A:** Firma Müller, guten Tag.
**Person B:** Guten Tag, hier ist Anna Schmidt. Ich bin krank.
**Person A:** Oh, was haben Sie?
**Person B:** Ich habe Fieber und Halsschmerzen. Ich kann heute nicht kommen.
**Person A:** Okay. Bleiben Sie zu Hause und werden Sie gesund!
**Person B:** Danke! Ich melde mich wieder, wenn ich gesund bin.

**الترجمة:**
**الشخص أ:** شركة مولر، يوم سعيد.
**الشخص ب:** يوم سعيد، أنا آنا شميت. أنا مريضة.
**الشخص أ:** أوه، ماذا بك؟
**الشخص ب:** عندي حرارة وألم في الحلق. لا أستطيع القدوم اليوم.
**الشخص أ:** حسناً. ابق في البيت وطيّب!
**الشخص ب:** شكراً! سأتصل مرة أخرى عندما أكون بخير.

## 4. ✍️ القاعدة بأبسط شكل

### 🟢 القاعدة 1: Ich habe + Symptom (عندي...)

| 🤒 الألم | 🗣️ الجملة |
|---------|----------|
| صداع | Ich habe **Kopfschmerzen**. |
| حمى | Ich habe **Fieber**. |
| سعال | Ich habe **Husten**. |
| زكام | Ich habe **Schnupfen**. |
| ألم بطن | Ich habe **Bauchschmerzen**. |
| ألم ظهر | Ich habe **Rückenschmerzen**. |

### 🟢 القاعدة 2: Mir tut ... weh (يؤلمني...)

| 🤕 الجزء | 🗣️ الجملة |
|---------|----------|
| الرأس | **Mir tut** der Kopf **weh**. |
| الحلق | **Mir tut** der Hals **weh**. |
| البطن | **Mir tut** der Bauch **weh**. |
| الظهر | **Mir tut** der Rücken **weh**. |
| العيون (جمع) | **Mir tun** die Augen **weh**. |
| الرجل (جمع) | **Mir tun** die Beine **weh**. |

### 🟢 القاعدة 3: müssen = يجب / مضطر

| Deutsch | العربية |
|---------|---------|
| Ich **muss** zum Arzt gehen. | **يجب** أن أذهب إلى الطبيب. |
| Ich **muss** zu Hause bleiben. | **يجب** أن أبقى في البيت. |
| Ich **muss** Medikamente nehmen. | **يجب** أن آخذ دواء. |

### 🟢 القاعدة 4: kann nicht = لا أستطيع

| Deutsch | العربية |
|---------|---------|
| Ich **kann** heute nicht arbeiten. | **لا أستطيع** العمل اليوم. |
| Ich **kann** nicht kommen. | **لا أستطيع** المجيء. |
| Ich **kann** nicht schlafen. | **لا أستطيع** النوم. |

### 🎯 الخلاصة — 4 جمل مهمة جداً:

| الموقف | الجملة الألمانية |
|--------|-----------------|
| 🟡 أقول أنا مريض | **Ich bin krank.** |
| 🔴 أقول عندي... | **Ich habe Fieber / Kopfschmerzen.** |
| 🟠 أقول يؤلمني... | **Mir tut der Kopf weh.** |
| 🟢 أقول لازم أروح للدكتور | **Ich muss zum Arzt gehen.** |

> 🚀 **نصيحة:** احفظ هالجمل الأربع وبتقدر تتكلم عن المرض في ألمانيا!

## 5. 🧠 تمارين سريعة

**التمرين 1: اختر الإجابة الصحيحة**

1. Ich ___ krank.
   - a) habe
   - b) bin
   - c) muss

2. Ich ___ Fieber.
   - a) bin
   - b) habe
   - c) tut

3. Mir ___ der Kopf weh.
   - a) habe
   - b) tut
   - c) tun

4. Ich ___ zum Arzt gehen.
   - a) habe
   - b) kann
   - c) muss

5. Mir ___ die Augen weh.
   - a) tut
   - b) habe
   - c) tun

✅ **الحل:** 1-b, 2-b, 3-b, 4-c, 5-c

---

**التمرين 2: اربط الجملة بالترجمة**

| # | Deutsch | # | Arabisch |
|---|---------|---|----------|
| 1 | Ich habe Fieber. | أ | أنا مريض. |
| 2 | Mir tut der Hals weh. | ب | عندي سعال. |
| 3 | Ich bin krank. | ج | عندي حمى. |
| 4 | Ich habe Husten. | د | حلقي يؤلمني. |
| 5 | Ich muss zum Arzt. | ه | يجب أن أذهب للطبيب. |

✅ **الحل:** 1-ج, 2-د, 3-أ, 4-ب, 5-ه
