## 1. 🎉 كلمات جديدة (Neue Wörter)

| **der Geburtstag** | **دير غيبورتستاغ** — deer geh-BOORTS-tahk | عيد الميلاد |

| **Neujahr** | **نوييار** — NOY-yahr | السنة الجديدة |
| **feiern** | **فايرن** — FYE-ern | يحتفل |
| **einladen** | **آينلادن** — AYN-lah-den | يدعو (يوجه دعوة) |
| **die Einladung** | **دي آينلادونغ** — dee AYN-lah-doong | الدعوة |
| **schenken** | **شينكن** — SHEN-ken | يهدي |
| **das Geschenk** | **داس غيشينك** — dahs geh-SHENK | الهدية |
| **bekommen** | **بيكومن** — beh-KOH-men | يستلم / ينال |
| **wünschen** | **فونشن** — VÜN-shen | يتمنى |

| **der Glückwunsch** | **دير غليكفونش** — deer GLÜK-vünsh | التهاني / التبريك |
| **der Kuchen** | **دير كوخن** — deer KOO-khen | الكعكة / الحلا |
| **die Kerze** | **دي كرتسه** — dee KER-tseh | الشمعة |

## 2. 📅 شهور السنة (Die Monate)

عشان تتكلم عن الأعياد، لازم تعرف شهور السنة بالألماني:

| الشهر | الألمانية | النطق |
|-------|-----------|-------|
| يناير | **Januar** | **يانوار** — YAH-noo-ahr |
| فبراير | **Februar** | **فيبروار** — FAY-broo-ahr |
| مارس | **März** | **ميرتس** — mehrts |
| أبريل | **April** | **أبريل** — ah-PREEL |
| مايو | **Mai** | **ماي** — my |
| يونيو | **Juni** | **يوني** — YOO-nee |
| يوليو | **Juli** | **يولي** — YOO-lee |
| أغسطس | **August** | **أوغست** — ow-GOOST |
| سبتمبر | **September** | **زپتمبر** — zep-TEM-ber |
| أكتوبر | **Oktober** | **أكتوبر** — ok-TOH-ber |
| نوفمبر | **November** | **نوفمبر** — no-VEM-ber |
| ديسمبر | **Dezember** | **دتسِمبر** — deh-TSEM-ber |

### 🎂 أعياد ميلاد المشاهير — أمثلة:

> **Mein Geburtstag ist im Mai.** — عيد ميلادي في مايو.
> 
> **Wann hast du Geburtstag?** — متى عيد ميلادك؟
> 
> **Ich habe am 15. März Geburtstag.** — عيد ميلادي في 15 مارس.

## 3. 📝 القاعدة: التهنئة والدعوة (Gratulieren und Einladen)

### 🎯 كيف تهنّئ أحد؟ (Glückwünsche)

في ألمانيا، فيه 3 طرق مشهورة للتهنئة:

| الألمانية | النطق | المعنى | متى تقولها |
|-----------|-------|--------|-----------|
| **Herzlichen Glückwunsch!** | **هيرتسليشن غليكفونش** — HAIRT-sli-chen GLÜK-vünsh | تهانينا الحارة / كل سنة وأنت طيب! | عيد ميلاد 🎂 |
| **Alles Gute zum Geburtstag!** | **آلس غوته تسوم غيبورتستاغ** — AH-les GOO-teh tsoom geh-BOORTS-tahk | كل التوفيق بمناسبة عيد ميلادك! | عيد ميلاد 🎂 |
| **Frohe Weihnachten!** | **فروه فايناختن** — FROH-eh VYE-nakh-ten | ميلاد مجيد! | الكريسماس 🎄 |
| **Guten Rutsch ins neue Jahr!** | **غوتن روتش إنس نويه يار** — GOO-ten rootsch ins NOY-eh yahr | سنة سعيدة! | رأس السنة 🎆 |
| **Frohe Ostern!** | **فروه أوسترن** — FROH-eh OH-stern | فصح سعيد! | عيد الفصح 🐣 |

### ✏️ كيف تكوّن جملة تهنئة؟

القاعدة سهلة جداً:

> **Ich wünsche dir + (شيء حلو)!**

| الألمانية | المعنى |
|-----------|--------|
| **Ich wünsche dir alles Gute!** | أتمنى لك كل الخير! |
| **Ich wünsche dir viel Glück!** | أتمنى لك حظاً سعيداً! |
| **Ich wünsche dir viel Spaß!** | أتمنى لك مرحاً كثيراً! |
| **Ich wünsche euch viel Freude!** | أتمنى لكم فرحاً كثيراً! (جمع) |

> **تذكير:** **dir** = لك (لشخص واحد صديق) — **euch** = لكم (لمجموعة) — **Ihnen** = لكم (رسمي)

### 🎈 كيف تدعو أحد؟ (Einladen)

عشان تدعي صديق لحفلة، استخدم هالجمل:

| الألمانية | المعنى |
|-----------|--------|
| **Ich lade dich ein!** | أدعوك! (غير رسمي) |
| **Ich lade Sie ein!** | أدعوكم! (رسمي) |
| **Kommst du zur Party?** | هل ستأتي إلى الحفلة؟ |
| **Wann ist die Party?** | متى الحفلة؟ |
| **Die Party ist am Samstag.** | الحفلة يوم السبت. |
| **Um wie viel Uhr?** | في أي ساعة؟ |
| **Um 18 Uhr.** | الساعة 6 مساءً. |

### 🎁 فعل schenken (يهدي)

| الضمير | التصريف | مثال |
|--------|---------|------|
| **ich** | **schenke** | Ich **schenke** dir eine Blume. |
| **du** | **schenkst** | Du **schenkst** mir ein Geschenk. |
| **er / sie** | **schenkt** | Er **schenkt** ihr einen Kuchen. |
| **wir** | **schenken** | Wir **schenken** ihm ein Buch. |
| **ihr** | **schenkt** | Ihr **schenkt** mir eine Kerze. |
| **sie / Sie** | **schenken** | Sie **schenken** uns Blumen. |

### 🎊 فعل feiern (يحتفل)

| الضمير | التصريف | مثال |
|--------|---------|------|
| **ich** | **feiere** | Ich **feiere** meinen Geburtstag. |
| **du** | **feierst** | Du **feierst** Weihnachten. |
| **er / sie** | **feiert** | Er **feiert** Silvester. |
| **wir** | **feiern** | Wir **feiern** zusammen. |
| **ihr** | **feiert** | Ihr **feiert** heute. |
| **sie / Sie** | **feiern** | Sie **feiern** Ostern. |

> 🎯 **القاعدة:** feiern فعل عادي — تضيف **-e, -st, -t, -en, -t, -en** مثل أغلب الأفعال!

## 4. 💬 محادثة (Gespräch)

> **Anna:** Hallo Tom! Ich habe am Samstag Geburtstag.  
> *(مرحباً توم! عندي عيد ميلاد يوم السبت.)*
> 
> **Tom:** Oh, wirklich? Herzlichen Glückwunsch! 🎂  
> *(أوه، حقاً؟ تهانينا الحارة!)*
> 
> **Anna:** Danke! Ich lade dich ein. Kommst du zur Party?  
> *(شكراً! أدعوك. هل ستأتي إلى الحفلة؟)*
> 
> **Tom:** Ja, gerne! Wann ist die Party?  
> *(نعم، بكل سرور! متى الحفلة؟)*
> 
> **Anna:** Am Samstag um 15 Uhr bei mir zu Hause.  
> *(يوم السبت الساعة 3 بعد الظهر عندي في البيت.)*
> 
> **Tom:** Super! Was kann ich mitbringen?  
> *(ممتاز! ماذا يمكنني أن أحضر معي؟)*
> 
> **Anna:** Du kannst einen Kuchen oder Saft mitbringen. 😊  
> *(تقدر تجيب كعكة أو عصير.)*
> 
> **Tom:** Alles klar! Ich bringe einen Kuchen mit. Und was wünschst du dir?  
> *(تمام! راح أجيب كعكة. وش تتمنى (لعيد ميلادك)؟)*
> 
> **Anna:** Ich wünsche mir Blumen! 💐  
> *(أتمنى ورد!)*
> 
> **Tom:** Ok! Bis Samstag! Tschüss!  
> *(أوكي! ليوم السبت! باي!)*
> 
> **Anna:** Tschüss! Bis bald!  
> *(باي! إلى اللقاء!)*

## 5. ✍️ تكوين الجمل (Sätze bilden)

### 🧩 جمل مفيدة عن الأعياد

| الألمانية | المعنى |
|-----------|--------|
| **Ich feiere meinen Geburtstag.** | أحتفل بعيد ميلادي. |
| **Wir feiern Weihnachten zu Hause.** | نحتفل بالكريسماس في البيت. |
| **Ich lade meine Freunde ein.** | أدعو أصدقائي. |
| **Meine Familie feiert zusammen.** | عائلتي تحتفل معاً. |
| **Ich bekomme viele Geschenke.** | أستلم هدايا كثيرة. |
| **Sie tanzen und singen.** | يرقصون ويغنون. |
| **Das Fest ist toll!** | الحفلة رائعة! |
| **Die Stimmung ist super!** | الجو رائع! |
| **Wir essen Kuchen.** | نأكل كعكة. |
| **Er zündet die Kerzen an.** | يشعل الشموع. |

### 🔄 ترتيب الجملة

في الألمانية **الفعل دائماً في المكان الثاني**:

| الترتيب الصحيح | الترتيب الخطأ |
|---------------|--------------|
| Ich **feiere** meinen Geburtstag. | ~~Ich meinen Geburtstag feiere.~~ ❌ |
| Am Samstag **habe** ich Geburtstag. | ~~Am Samstag ich habe Geburtstag.~~ ❌ |
| Wir **feiern** zusammen Weihnachten. | ~~Wir zusammen Weihnachten feiern.~~ ❌ |

## 6. 🎯 تمارين سريعة (Schnelle Übungen)

**Ü 1:** اكتب الكلمة الصحيحة:

| العربية | الألمانية |
|---------|-----------|
| عيد الميلاد | _______________ |
| الهدية | _______________ |
| يدعو | _______________ |
| يهدي | _______________ |
| يهنّئ | _______________ |

✅ **الحل:** der Geburtstag / das Geschenk / einladen / schenken / gratulieren

**Ü 2:** اختر التصريف الصحيح:

1. Ich ________ (feiern) meinen Geburtstag. → **feiere**
2. Du ________ (schenken) mir eine Blume. → **schenkst**
3. Er ________ (einladen) seine Freunde ein. → **lädt ein**
4. Wir ________ (tanzen) auf der Party. → **tanzen**
5. Ihr ________ (feiern) Silvester. → **feiert**

## 8. 🇩🇪 ثقافة: الأعياد في ألمانيا (Feste in Deutschland)

| العيد | التاريخ | ماذا يفعلون؟ |
|-------|---------|-------------|
| **Weihnachten** 🎄 | 24-26 ديسمبر | تجتمع العائلة، يتبادلون الهدايا، يأكلون كعكة الكريسماس |
| **Silvester / Neujahr** 🎆 | 31 ديسمبر | ألعاب نارية، يحتفلون في الشارع، يتمنون سنة سعيدة |
| **Ostern** 🐣 | مارس/أبريل (متحرك) | يدهنون البيض، يخبئون الحلوى للأطفال |
| **Geburtstag** 🎂 | حسب تاريخ ميلادك | حفلة، كعكة، شموع، هدايا، أغنية "Zum Geburtstag viel Glück" |
| **Oktoberfest** 🍺 | سبتمبر/أكتوبر | مهرجان في ميونخ — أشهر مهرجان في العالم! |

> 🇩🇪 **معلومة:** في ألمانيا، إذا كان عيد ميلادك يوم **أحد** أو **عطلة** — أحياناً تحتفل في اليوم اللي قبله أو بعده مش في نفس اليوم!
