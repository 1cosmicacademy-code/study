# صيغة الأمر (Imperativ) في اللغة الألمانية

🎯 **صيغة الأمر** نستخدمها لإعطاء أوامر، نصائح، أو طلبات. في الألمانية، لكل ضمير (du, ihr, Sie) صيغة أمر مختلفة.

---

## 💡 صيغ الأمر الثلاث

### 1. الأمر مع du — صيغة غير رسمية مفرد

**القاعدة:** تصريف du - حذف -st + أحياناً حذف علامة الاستفهام (Umlaut)

| الفعل | du Präsens | الأمر |
|-------|-----------|-------|
| **kommen** | du kommst | **Komm!** (تعال!) |
| **gehen** | du gehst | **Geh!** (اذهب!) |
| **spielen** | du spielst | **Spiel!** (العب!) |
| **warten** | du wartest | **Warte!** (انتظر!) ← تنتهي بـ -est |
| **lesen** | du liest | **Lies!** (اقرأ!) ← يتغير |
| **sehen** | du siehst | **Sieh!** (انظر!) |
| **fahren** | du fährst | **Fahr!** (سق/اذهب!) ← بدون Umlaut |

### 2. الأمر مع ihr — صيغة غير رسمية جمع

**القاعدة:** نفس تصريف ihr بالضبط

| الفعل | ihr Präsens | الأمر |
|-------|-----------|-------|
| **kommen** | ihr kommt | **Kommt!** (تعالوا!) |
| **gehen** | ihr geht | **Geht!** (اذهبوا!) |
| **spielen** | ihr spielt | **Spielt!** (العبوا!) |
| **lesen** | ihr lest | **Lest!** (اقرأوا!) |

### 3. الأمر مع Sie — صيغة رسمية (مفرد وجمع)

**القاعدة:** تصريف Sie + الفعل + Sie

| الفعل | Sie Präsens | الأمر |
|-------|-----------|-------|
| **kommen** | Sie kommen | **Kommen Sie!** (تفضل!) |
| **gehen** | Sie gehen | **Gehen Sie!** (اذهب!) |
| **sein** | Sie sind | **Seien Sie!** (كن/كونوا!) |

## ⚠️ الأفعال الشاذة في الأمر

### sein — يكون (شاذ تماماً)

| الصيغة | الأمر |
|--------|-------|
| du | **Sei!** (كن!) |
| ihr | **Seid!** (كونوا!) |
| Sie | **Seien Sie!** (كن/كونوا!) |

### haben — يملك

| الصيغة | الأمر |
|--------|-------|
| du | **Hab!** (املك!) |
| ihr | **Habt!** (املكوا!) |
| Sie | **Haben Sie!** (املك!) |

### werden — يصير

| الصيغة | الأمر |
|--------|-------|
| du | **Werde!** (صِر!) |
| ihr | **Werdet!** (صيروا!) |
| Sie | **Werden Sie!** (صِر!) |

## 🎯 قاعدة ذهبية

```
du → خذ تصريف du واحذف -st (أحياناً Umlaut يختفي)
ihr → نفس تصريف ihr بالضبط
Sie → فعل + Sie (معكوس الترتيب)

Komm! Geht! Kommen Sie!
```
