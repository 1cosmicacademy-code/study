## 🎯 أهداف الدرس

- فهم استعمال الفعلين **sein** و **haben** كأفعال أساسية ومساعدة.
- التعرف على مجموعة الأفعال الخاصة (werden, bleiben, gehen, kommen) وكيفية ربطها بالحالات النحوية.
- تطبيق القواعد على جمل حياة يومية.

---

## 📚 القاعدة الأساسية

### 1️⃣ فعل **sein** (to be) – فعل ربط

| الفاعل | الحاضر (Präsens) |
|--------|-------------------|
| ich    | bin               |
| du     | bist              |
| er/sie/es | ist          |
| wir    | sind              |
| ihr    | seid              |
| sie/Sie| sind              |

- يستخدم **sein** لتحديد حالة الشخص أو الشيء (الهوية، المكان، الوقت).
- لا تحتاج أداة جر، وإنما تتبع مباشرة بالاسم أو الصفة.

### 2️⃣ فعل **haben** (to have) – فعل ملكية

| الفاعل | الحاضر (Präsens) |
|--------|-------------------|
| ich    | habe              |
| du     | hast              |
| er/sie/es | hat           |
| wir    | haben             |
| ihr    | habt              |
| sie/Sie| haben             |

- يستخدم **haben** للإشارة إلى الملكية أو كفعل مساعد مع الأفعال المركبة (Perfekt).

### 3️⃣ الأفعال الخاصة (besondere Verben)

| الفعل | المعنى | مثال الاستخدام |
|-------|--------|------------------|
| werden | يصبح | Ich **werde** müde. — أُصبح متعبًا. |
| bleiben | يبقى | Er **bleibt** zu Hause. — هو يبقى في البيت. |
| gehen | يذهب | Wir **gehen** ins Kino. — نذهب إلى السينما. |
| kommen | يأتِي | Sie **kommt** aus Berlin. — هي تأتي من برلين. |

هذه الأفعال تتصرف عادةً مع **Akkusativ** أو **Dativ** بحسب الفعل والظرف.

## ✅ جمل أمثلة مع الترجمة

| الجملة الألمانية | الترجمة العربية |
|-------------------|-------------------|
| Ich **bin** Student. | أنا طالب. |
| Du **bist** müde. | أنت متعب. |
| Er **hat** ein Buch. | هو يملك كتابًا. |
| Wir **haben** keine Zeit. | ليس لدينا وقت. |
| Sie **wird** Ärztin. | هي ستصبح طبيبة. |
| Ich **bleibe** hier. | أَبقى هنا. |
| Wir **gehen** zum Markt. | نذهب إلى السوق. |
| Ihr **kommt** morgen. | ستأتون غدًا. |
