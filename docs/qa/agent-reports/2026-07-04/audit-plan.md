## 5. معايير القبول والرفض

### أخطاء تمنع إصدار التقرير النهائي (Blockers):
- الموقع لا يعمل (`mkdocs serve` يفشل)
- أخطاء JavaScript تمنع تحميل الصفحة
- اختبار المستوى (42 سؤالاً) لا يعمل
- جميع تمارين A1 Übungsbuch لا تعمل

### أخطاء تُسجل ولا تمنع التقرير:
- أخطاء UI تجميلية
- روابط مقطوعة في صفحات غير رئيسية
- أخطاء إملائية فردية
- مشاكل أداء بسيطة
