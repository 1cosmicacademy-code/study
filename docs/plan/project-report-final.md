## 4. مكونات النظام

### 4.1 `video-switcher.js` — سكربت تبديل الفيديو

**الدور:** يكتشف لغة الصفحة (`/en/` vs `/`)، ويستخرج رقم الدرس (`lektion-XX`)، ويجلب `videos.json`، ثم يُنشئ iframe يوتيوب تلقائياً بعد عنوان h2 المناسب.

**آلية العمل:**
```
Page Load → detect language → extract lesson key → fetch videos.json
→ find h2 containing "فيديو"/"Video" → create/update iframe embed
→ update fallback text links and code blocks
```

**الحالات التي يعالجها:**
- ✅ صفحة عربية (بدون `/en/`) — يستخدم video_id العربي
- ✅ صفحة إنجليزية (بـ `/en/`) — يستخدم video_id الإنجليزي
- ✅ تحديث iframe موجود بدل إنشاء new container
- ✅ روابط نصية و code blocks يتم تحديثها كـ fallback
- ✅ فشل silent في حال عدم وجود `videos.json` أو مفتاح الدرس

### 4.2 `videos.json` — قاعدة بيانات الفيديوهات

**البنية:**
```json
{
  "lektion-01": {
    "en": {
      "video_id": "iB_sassbnOw",
      "title": "Learn German | Greetings | German for beginners | A1 - Lesson 1",
      "channel": "Learn German with Anja",
      "description": "...",
      "playlist": "PLxxx"
    },
    "ar": {
      "video_id": "kolo85cQdxo",
      "title": "تعلم الألمانية A1 - الدرس 1: التحيات والمقدمة",
      "channel": "Deutsch mit Mira",
      "description": "..."
    }
  }
}
```

## 5. تنسيق صفحة الدرس القياسي

كل صفحة درس (بعد الإصلاح) تتبع هذا التنسيق:

```markdown
# Lektion XX — ...

## 🎬 فيديو الدرس    ← [عربي] / ## 🎬 Lesson Video [إنجليزي]
[وصف الفيديو]
[video-switcher.js يُنشئ iframe تلقائياً هنا]

[باقي محتوى الدرس]
```

**ملاحظة:** Lektion 08 تستخدم `🎬 فيديو يوتيوب` بدلاً من `🎬 فيديو الدرس` — و video-switcher.js يعالج كلاهما.

## 6. مشاكل وحلول

| المشكلة | الحل |
|---------|------|
| 36 صفحة درس تستخدم `<blockquote>` + `<iframe>` ثابت | تحويل إلى `## h2` + وصف — video-switcher.js ينشئ iframe ديناميكياً |
| iframe ثابت بلغة واحدة فقط في 36 صفحة | إزالة الـ iframe الثابت — السكربت يختار الفيديو حسب اللغة |
| عناوين دروس تبدأ بخط غير ثابت (Lektion 01: `#` vs الباقي `##`) | توحيد كل العناوين إلى `#` للعنوان الرئيسي و`##` للفيديو |
| اختبار التوقيت — `browser_evaluate` يجري قبل أن ينتهي video-switcher.js | انتظر 3–5 ثوانٍ قبل فحص وجود الفيديو |
| sitemap.xml 404 في console | ظاهرة طبيعية مع static-i18n — لا تأثير لها |
