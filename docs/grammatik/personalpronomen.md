# الضمائر الشخصية (Personalpronomen) في اللغة الألمانية

🎯 **الضمائر الشخصية** تحل محل الأسماء — بدل ما تكرر الاسم، تستخدم ضميراً: ich, du, er, sie, es...

---

## 💡 جدول الضمائر في Nominativ

| الشخص | الضمير | المعنى | مثال |
|--------|-------|--------|------|
| 1st مفرد | **ich** | أنا | Ich bin hier. |
| 2nd مفرد | **du** | أنت (قريب/غير رسمي) | Du bist nett. |
| 3rd مذكر | **er** | هو | Er kommt. |
| 3rd مؤنث | **sie** | هي | Sie singt. |
| 3rd محايد | **es** | هو/هي (للمحايد) | Es regnet. |
| 1st جمع | **wir** | نحن | Wir lernen. |
| 2nd جمع | **ihr** | أنتم (جمع غير رسمي) | Ihr seid lustig. |
| 3rd جمع | **sie** | هم | Sie tanzen. |
| رسمي (مف/ج) | **Sie** | حضرتك/حضرتكم (رسمي) | Sie sind Herr Müller. |

> **ملاحظة:** الضمير الرسمي **Sie** يكتب دائماً بحرف كبير، ويستخدم للمفرد والجمع بنفس الصيغة.

## ⚠️ استخدامات مهمة

### الفرق بين du / ihr / Sie

| الضمير | متى نستخدم؟ | مثال |
|--------|------------|------|
| **du** | مع شخص واحد قريب (صديق، طفل، قريب) | **Du** bist mein Freund. |
| **ihr** | مع مجموعة قريبة | **Ihr** seid meine Freunde. |
| **Sie** | رسمي (مع غرباء، في العمل، مع الكبار) | **Sie** sind Herr Schmidt. |

### الضمير "es" — استخدامات خاصة

1. **كضمير شخصي:** Das Kind weint. **Es** ist traurig. (الطفل يبكي. هو حزين.)
2. **مع الطقس:** **Es** regnet. (إنها تمطر.)
3. **مع الزمن:** **Es** ist 5 Uhr. (الساعة 5.)
4. **كفاعل وهمي:** **Es** gibt ein Problem. (هناك مشكلة.)

## 📌 قاعدة ذهبية

```
ich → أنا (المتكلم)
du → أنت (المخاطب قريب)
er/sie/es → هو/هي (الغائب)
wir → نحن
ihr → أنتم (جمع مخاطب)
sie → هم (جمع غائب)
Sie → حضرتك (رسمي — بحرف كبير!)
```

## ✅ أمثلة مع الترجمة

| الجملة | الترجمة |
|--------|---------|
| **Ich** heiße Ali. | اسمي علي. |
| **Du** bist fleißig. | أنت مجتهد. |
| **Er** ist mein Bruder. | هو أخي. |
| **Sie** ist meine Schwester. | هي أختي. |
| **Es** ist klein. | هو صغير. |
| **Wir** kommen aus Ägypten. | نحن من مصر. |
| **Ihr** habt ein Haus. | أنتم عندكم منزل. |
| **Sie** wohnen in Berlin. | هم يسكنون في برلين. |
| **Sie** (رسمي) sind Herr Doktor. | حضرتك الدكتور. |

## 📝 تمارين

### تمرين 1: أكمل بالضمير الصحيح

1. ___ heiße Anna. (أنا)
2. ___ bist mein Freund. (أنت)
3. ___ ist meine Mutter. (هي)
4. ___ sind Ärzte. (نحن)
5. ___ seid laut. (أنتم — جمع)

### تمرين 2: استبدل الاسم بالضمير

1. Der Mann arbeitet. → ___ arbeitet.
2. Die Frau lacht. → ___ lacht.
3. Das Kind spielt. → ___ spielt.
4. Die Kinder tanzen. → ___ tanzen.
5. Anna und ich lesen. → ___ lesen.

### تمرين 3: du, ihr oder Sie?

1. Zum Arzt: "___ sind Herr Müller?"
2. Zum Freund: "___ kommst morgen?"
3. Zu den Kindern: "___ spielt im Garten?"
4. Zur Kollegin (رسمي): "Kommen ___ morgen?"
5. Zur Gruppe Freunde: "___ seid cool!"
