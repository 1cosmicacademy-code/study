## التمرين 9: اكتب الحوار ✍️

**أكمل الحوار بين شخصين:**

| المتحدث | النص | الإجابة |

| B | Oh, wirklich? Herzlichen ___________________! 🎂 *(أوه، حقاً؟ تهانينا الحارة!)* | |

| B | Super! Ich bringe einen ___________________ mit. *(ممتاز! سأحضر كعكة.)* | |
