# الأدوات (Artikel) في اللغة الألمانية — der, die, das, ein, keine

🎯 **أدوات التعريف** في الألمانية تخبرنا عن جنس الاسم (مذكر، مؤنث، محايد) وعدده (مفرد، جمع). أهم قاعدة: **احفظ الاسم مع أداته!**

## 💡 أنواع الأدوات

### 1. أدوات التعريف (Definite Articles) — "the"

| الأداة | الاستخدام | مثال |
|-------|----------|------|
| **der** | مذكر (m) | der Mann (الرجل) |
| **die** | مؤنث (f) | die Frau (المرأة) |
| **das** | محايد (n) | das Kind (الطفل) |
| **die** | جمع (pl) | die Kinder (الأطفال) |

### 2. أدوات التنكير (Indefinite Articles) — "a/an"

| الأداة | الاستخدام | مثال |
|-------|----------|------|
| **ein** | مذكر | ein Mann (رجل) |
| **eine** | مؤنث | eine Frau (امرأة) |
| **ein** | محايد | ein Kind (طفل) |
| **---** | جمع | --- (لا يوجد) |

### 3. النفي بـ kein — "no/not a"

| الأداة | الاستخدام | مثال |
|-------|----------|------|
| **kein** | مذكر | kein Mann (لا رجل) |
| **keine** | مؤنث | keine Frau (لا امرأة) |
| **kein** | محايد | kein Kind (لا طفل) |
| **keine** | جمع | keine Kinder (لا أطفال) |

## ⚠️ كيف تعرف جنس الاسم؟

لا توجد قاعدة ثابتة 100%، لكن هناك أنماط تساعدك:

**مذكر (der) عادةً:**
- الذكور: der Mann, der Vater, der Junge
- أيام الأسبوع: der Montag, der Dienstag
- الشهور: der Januar, der Mai
- الفصول: der Sommer, der Winter
- النهايات: -er, -ig, -ling, -or

**مؤنث (die) عادةً:**
- الإناث: die Frau, die Mutter, die Schwester
- النهايات: -ung, -heit, -keit, -schaft, -ion, -tät, -ik
- الأرقام: die Eins, die Zwei
- الأشجار والزهور: die Rose, die Eiche

**محايد (das) عادةً:**
- الأطفال/الصغار: das Kind, das Baby
- النهايات: -chen, -lein (تصغير)
- المصادر (فعل → اسم): das Essen, das Trinken
- المعادن: das Gold, das Silber
- النهايات: -um, -ment, -tum

## 📌 قاعدة ذهبية

```
احفظ كل اسم مع أداته! 
ليس "Tisch" بل "der Tisch"
ليس "Lampe" بل "die Lampe"
ليس "Buch" بل "das Buch"
```

## ✅ جدول شامل

| النوع | معرف (the) | نكرة (a) | نفي (no) |
|------|-----------|---------|---------|
| مذكر | der | ein | kein |
| مؤنث | die | eine | keine |
| محايد | das | ein | kein |
| جمع | die | (kein) | keine |
