# تصريف الأفعال في المضارع (Verbkonjugation — Präsens)

🎯 **تصريف الأفعال** يعني تغيير الفعل حسب الفاعل. في الألمانية، كل ضمير له نهاية خاصة تضاف للفعل.

---

## 💡 النهايات القياسية للأفعال العادية

معظم الأفعال الألمانية تتبع نفس النمط. الفعل = الجذر + النهاية:

| الضمير | النهاية | مثال (spielen — يلعب) |
|--------|---------|----------------------|
| **ich** | **-e** | ich spiel**e** |
| **du** | **-st** | du spiel**st** |
| **er/sie/es** | **-t** | er spiel**t** |
| **wir** | **-en** | wir spiel**en** |
| **ihr** | **-t** | ihr spiel**t** |
| **sie/Sie** | **-en** | sie spiel**en** |

**أمثلة:**
```
lernen (يتعلم): ich lerne, du lernst, er lernt, wir lernen, ihr lernt, sie lernen
wohnen (يسكن): ich wohne, du wohnst, er wohnt, wir wohnen, ihr wohnt, sie wohnen
```

## ⚠️ الفعلين المساعدين: sein (يكون) و haben (يملك)

هذين الفعلين **شاذين** ويجب حفظهما عن ظهر قلب:

| الضمير | **sein** (يكون) | **haben** (يملك) |
|--------|----------------|-----------------|
| **ich** | **bin** | **habe** |
| **du** | **bist** | **hast** |
| **er/sie/es** | **ist** | **hat** |
| **wir** | **sind** | **haben** |
| **ihr** | **seid** | **habt** |
| **sie/Sie** | **sind** | **haben** |

**أمثلة:**
- Ich **bin** Ali. — أنا علي.
- Du **bist** fleißig. — أنت مجتهد.
- Er **ist** Lehrer. — هو مدرس.
- Wir **haben** ein Haus. — نحن نملك منزلاً.

## ⚠️ أفعال شاذة أخرى

### الأفعال التي تتغير فيها العلة (Vokalwechsel)
بعض الأفعال تغير حرف العلة في du + er/sie/es:

| الفعل | المعنى | du | er/sie/es |
|-------|-------|----|-----------|
| **fahren** | يقود/يذهب | f**ä**hrst | f**ä**hrt |
| **sehen** | يرى | s**ie**hst | s**ie**ht |
| **lesen** | يقرأ | l**ie**st | l**ie**st |
| **sprechen** | يتكلم | spr**i**chst | spr**i**cht |
| **essen** | يأكل | **i**sst | **i**sst |
| **nehmen** | يأخذ | n**i**mmst | n**i**mmt |

### الأفعال المنتهية بـ -eln و -ern
- **lächeln** (يبتسم): ich läch**le**, du lächelst, er lächelt
- **ändern** (يغير): ich änd**re**, du änderst, er ändert

## 📌 قاعدة ذهبية

```
ich → -e      wir → -en
du → -st      ihr → -t
er/sie/es → -t  sie/Sie → -en

sein: bin, bist, ist, sind, seid, sind  ← احفظها!
haben: habe, hast, hat, haben, habt, haben
```
