# 📝 تمارين Lektion 4: Familie!

> 🎯 **الهدف:** اختبر نفسك في أفراد العائلة وصفات الملكية (mein / meine / dein / seine / ihr / unser / euer).  
> 💡 **نصيحة:** حاول تحل التمارين بنفسك أولاً، بعدين شوف الحل!

---

## التمرين 1: املأ الفراغ — أفراد العائلة 🖊️

**شوف الصورة (في خيالك): هذي عائلة محمد. أكمل اسم الشخص المناسب:**

| الجملة | اسم الشخص بالألماني |
|--------|-------------------|
| 1. محمد هو **_________** (الأب). | |
| 2. فاطمة هي **_________** (الأم). | |
| 3. أحمد هو **_________** من محمد وفاطمة. | |
| 4. مريم هي **_________** من محمد وفاطمة. | |
| 5. عباس هو **_________** (جد) أحمد ومريم. | |
| 6. زينب هي **_________** (جدة) أحمد ومريم. | |

✅ **الحل:**

| # | الإجابة |
|---|---------|
| 1 | **der Vater** |
| 2 | **die Mutter** |
| 3 | **der Sohn** |
| 4 | **die Tochter** |
| 5 | **der Opa / der Großvater** |
| 6 | **die Oma / die Großmutter** |

## التمرين 2: اختر mein أو meine 🎯

**اختار: mein ولا meine؟**

| # | الجملة | mein | meine |
|---|--------|:----:|:-----:|
| 1 | Das ist _____ Vater. | ⬜ | ⬜ |
| 2 | Das ist _____ Mutter. | ⬜ | ⬜ |
| 3 | Das ist _____ Bruder. | ⬜ | ⬜ |
| 4 | Das ist _____ Schwester. | ⬜ | ⬜ |
| 5 | Das ist _____ Kind. | ⬜ | ⬜ |
| 6 | Das sind _____ Kinder. | ⬜ | ⬜ |
| 7 | Das ist _____ Haus. | ⬜ | ⬜ |
| 8 | Das ist _____ Oma. | ⬜ | ⬜ |

✅ **الحل:**

| # | الإجابة | الشرح |
|---|---------|-------|
| 1 | **mein** | Vater مذكر (der) → mein |
| 2 | **meine** | Mutter مؤنث (die) → meine |
| 3 | **mein** | Bruder مذكر (der) → mein |
| 4 | **meine** | Schwester مؤنث (die) → meine |
| 5 | **mein** | Kind محايد (das) → mein |
| 6 | **meine** | Kinder جمع (die) → meine |
| 7 | **mein** | Haus محايد (das) → mein |
| 8 | **meine** | Oma مؤنث (die) → meine |

## التمرين 3: صِل (Match) — صفات الملكية 🔗

**صِل صاحب الملكية بكلمة الملكية الصحيحة:**

| صاحب الملكية | كلمة الملكية |
|:------------:|:------------:|
| 1. ich (أنا) | أ. sein |
| 2. du (أنت) | ب. euer |
| 3. er (هو) | ج. mein |
| 4. sie (هي) | د. unser |
| 5. wir (نحن) | ه. dein |
| 6. ihr (أنتم) | و. ihr |

✅ **الحل:**

| # | الإجابة |
|---|---------|
| 1 | **ج** — ich → mein |
| 2 | **ه** — du → dein |
| 3 | **أ** — er → sein |
| 4 | **و** — sie (هي) → ihr |
| 5 | **د** — wir → unser |
| 6 | **ب** — ihr (أنتم) → euer |

## التمرين 4: املأ الفراغ — صفات الملكية ✍️

**اكتب كلمة الملكية الصحيحة (mein / meine / dein / deine / sein / seine / ihr / ihre / unser / unsere / euer / eure):**

| # | الجملة | الإجابة |
|---|--------|---------|
| 1 | Ich habe einen Bruder. _____ Bruder heißt Max. | ________ |
| 2 | Du hast eine Schwester. _____ Schwester ist klein. | ________ |
| 3 | Er hat einen Hund. _____ Hund ist süß! | ________ |
| 4 | Sie (هي) hat eine Katze. _____ Katze ist schwarz. | ________ |
| 5 | Wir haben ein Haus. _____ Haus ist groß. | ________ |
| 6 | Ihr (أنتم) habt ein Buch. _____ Buch ist neu. | ________ |
| 7 | Ich habe eine Mutter. _____ Mutter ist nett. | ________ |
| 8 | Du hast einen Opa. _____ Opa ist jung. | ________ |
| 9 | Sie (هم) haben Kinder. _____ Kinder spielen. | ________ |
| 10 | Wir haben eine Tante. _____ Tante heißt Anna. | ________ |

✅ **الحل:**

| # | الإجابة | الشرح |
|---|---------|-------|
| 1 | **Mein** | ich → mein + Bruder (مذكر) → mein |
| 2 | **Deine** | du → dein + Schwester (مؤنث) → deine |
| 3 | **Sein** | er → sein + Hund (مذكر) → sein |
| 4 | **Ihre** | sie (هي) → ihr + Katze (مؤنث) → ihre |
| 5 | **Unser** | wir → unser + Haus (محايد) → unser |
| 6 | **Euer** | ihr (أنتم) → euer + Buch (محايد) → euer |
| 7 | **Meine** | ich → mein + Mutter (مؤنث) → meine |
| 8 | **Dein** | du → dein + Opa (مذكر) → dein |
| 9 | **Ihre** | sie (هم) → ihr + Kinder (جمع) → ihre |
| 10 | **Unsere** | wir → unser + Tante (مؤنث) → unsere |

## التمرين 5: صحح الخطأ ❌

**في كل جملة خطأ. صححه:**

| # | الجملة (غلط) | التصحيح |
|---|--------------|---------|
| 1 | Das ist **mein** Mutter. | ______________ |
| 2 | Das ist **deine** Vater. | ______________ |
| 3 | Das ist **seine** Bruder. | ______________ |
| 4 | Das ist **unser** Oma. | ______________ |
| 5 | Das ist **eure** Kind. | ______________ |
| 6 | Das sind **mein** Kinder. | ______________ |
| 7 | Das ist **ihre** Onkel. | ______________ |
| 8 | Das ist **deine** Haus. | ______________ |

✅ **الحل:**

| # | التصحيح | الشرح |
|---|---------|-------|
| 1 | **meine** Mutter | Mutter مؤنث ← meine (مو mein) |
| 2 | **dein** Vater | Vater مذكر ← dein (مو deine) |
| 3 | **seinen** ← **sein** Bruder | Bruder مذكر ← sein (مو seine) |
| 4 | **unsere** Oma | Oma مؤنث ← unsere (مو unser) |
| 5 | **euer** Kind | Kind محايد ← euer (مو eure) |
| 6 | **meine** Kinder | Kinder جمع ← meine (مو mein) |
| 7 | **ihr** Onkel | Onkel مذكر ← ihr (مو ihre) |
| 8 | **dein** Haus | Haus محايد ← dein (مو deine) |

## التمرين 6: ترجم (Translate) 🌍

**ترجم هذه الجمل من العربية إلى الألمانية:**

| # | العربية | الألمانية |
|---|---------|-----------|
| 1 | هذا والدي. | ______________ |
| 2 | هذه أختي. | ______________ |
| 3 | هل هذا أخوك؟ | ______________ |
| 4 | نعم، هذا كلبي. | ______________ |
| 5 | أختك صغيرة. | ______________ |
| 6 | هذا بيتنا. | ______________ |
| 7 | جدتي لطيفة. | ______________ |
| 8 | أطفالهم يلعبون. | ______________ |

✅ **الحل:**

| # | الألمانية | ملاحظات |
|---|-----------|---------|

| 2 | **Das ist meine Schwester.** | Schwester مؤنث → meine |

| 5 | **Deine Schwester ist klein.** | Schwester مؤنث → deine |
| 6 | **Das ist unser Haus.** | Haus محايد → unser |
| 7 | **Meine Oma ist nett / süß.** | Oma مؤنث → meine |

## التمرين 7: اختر الإجابة الصحيحة 🎯

**اختر كلمة الملكية الصحيحة:**

| # | الجملة | أ | ب | ج |
|---|--------|---|---|---|
| 1 | Ich habe _____ Schwester. | mein | **meine** | meinen |
| 2 | Du hast _____ Bruder. | deine | **dein** | deinen |
| 3 | Er hat _____ Hund. | seine | seinem | **sein** |
| 4 | Sie (هي) hat _____ Katze. | ihr | **ihre** | ihren |
| 5 | Wir haben _____ Haus. | **unser** | unsere | unseren |
| 6 | Ihr habt _____ Oma. | euer | **eure** | euren |
| 7 | _____ Vater ist groß. (ich) | **Mein** | Meine | Meinen |
| 8 | _____ Kinder sind klein. (wir) | Unser | **Unsere** | Unseren |

✅ **الحل:**

| # | الإجابة | الشرح |
|---|---------|-------|
| 1 | **ب (meine)** | Schwester مؤنث → meine |
| 2 | **ب (dein)** | Bruder مذكر → dein |
| 3 | **ج (sein)** | Hund مذكر → sein |
| 4 | **ب (ihre)** | Katze مؤنث → ihre |
| 5 | **أ (unser)** | Haus محايد → unser |
| 6 | **ب (eure)** | Oma مؤنث → eure |
| 7 | **أ (Mein)** | Vater مذكر → mein |
| 8 | **ب (Unsere)** | Kinder جمع → unsere |

## التمرين 9: اكتب الحوار ✍️

**أكمل الحوار بكلمات الملكية الصحيحة:**

> **A:** Hallo! Wer ist das?  
> **B:** Das ist _____ (mein / meine) Familie!  
> **A:** Ist das _____ (dein / deine) Vater?  
> **B:** Ja, das ist _____ (mein / meine) Vater. Er heißt Sami.  
> **A:** Und ist das _____ (dein / deine) Mutter?  
> **B:** Ja, das ist _____ (mein / meine) Mutter. Sie heißt Lina.  
> **A:** Hast du Geschwister?  
> **B:** Ja, ich habe _____ (ein / eine) Bruder und _____ (ein / eine) Schwester.  
> **A:** Wie heißt _____ (dein / deine) Bruder?  
> **B:** _____ (Mein / Meine) Bruder heißt Ali.  
> **A:** Und _____ (dein / deine) Schwester?  
> **B:** _____ (Mein / Meine) Schwester heißt Sara.

✅ **الحل:**

> **A:** Hallo! Wer ist das?  
> **B:** Das ist **meine** Familie! *(Familie مؤنث → meine)*  
> **A:** Ist das **dein** Vater? *(Vater مذكر → dein)*  
> **B:** Ja, das ist **mein** Vater. Er heißt Sami. *(Vater مذكر → mein)*  
> **A:** Und ist das **deine** Mutter? *(Mutter مؤنث → deine)*  
> **B:** Ja, das ist **meine** Mutter. Sie heißt Lina. *(Mutter مؤنث → meine)*  
> **A:** Hast du Geschwister?  
> **B:** Ja, ich habe **einen** Bruder und **eine** Schwester.  
> **A:** Wie heißt **dein** Bruder? *(Bruder مذكر → dein)*  
> **B:** **Mein** Bruder heißt Ali. *(Bruder مذكر → mein)*  
> **A:** Und **deine** Schwester? *(Schwester مؤنث → deine)*  
> **B:** **Meine** Schwester heißt Sara. *(Schwester مؤنث → meine)*
