# Passive Voice (Passiv) in German

🎯 **Passiv** is used when **the action is important**, not who does it. For example, "The door is opened" (we don't care who opened it) instead of "Ahmed opened the door."

---

## 💡 Difference between Aktiv and Passiv

| Aktiv (Active) | Passiv (Passive) |
|----------------|------------------|
| The doer performs the action | The action happens — the doer is unimportant or unknown |
| **The chef** cooks the soup. | The soup **is cooked**. |
| **Der Koch** kocht die Suppe. | Die Suppe **wird gekocht**. |

---

## ⚠️ Forming Passiv

### Basic rule:
> **werden (conjugated) + Partizip II (at the end of the sentence)**

```
[Subject] + werden (conjugated) + ... + Partizip II
```

**Examples:**
- `Das Essen **wird gekocht**.` — The food is cooked.
- `Die Tür **wird geöffnet**.` — The door is opened.
- `Die Hausaufgaben **werden gemacht**.` — The homework is done.

### Conjugation of werden in Passiv (Present):

| Pronoun | werden + Partizip II |
|---------|----------------------|
| **ich** | werde gefragt |
| **du** | wirst gefragt |
| **er/sie/es** | wird gefragt |
| **wir** | werden gefragt |
| **ihr** | werdet gefragt |
| **sie/Sie** | werden gefragt |

---

## 🎯 Steps to convert Aktiv → Passiv

Here's a **simple plan** for conversion:

| Step | Aktiv | Passiv |
|------|-------|--------|
| 1️⃣ | **The chef** cooks the soup. | Find the subject (Der Koch) |
| 2️⃣ | The chef cooks **the soup**. | → **Die Suppe** becomes Subject (Nominativ) |
| 3️⃣ | The chef cooks the soup. | → The verb becomes **wird gekocht** |
| 4️⃣ | ✂️ The chef | → Old subject is removed or becomes **von + Dativ** |
| ✅ | **Die Suppe** wird (von dem Koch) **gekocht**. | Final result |

**Another example:**
- Aktiv: `Der Lehrer erklärt die Grammatik.` — The teacher explains grammar.
- Passiv: `Die Grammatik **wird** (vom Lehrer) **erklärt**.` — Grammar is explained (by the teacher).

---

## ⚠️ Passiv with Modalverben

When combining Modalverb with Passiv:

```
[Subject] + Modalverb (conj.) + Partizip II + werden (at the very end!)
```

| Modalverb | Aktiv | Passiv |
|-----------|-------|--------|
| **müssen** | Man muss die Tür schließen. | Die Tür **muss geschlossen werden**. |
| **können** | Man kann das Fenster öffnen. | Das Fenster **kann geöffnet werden**. |
| **sollen** | Man soll das Licht ausschalten. | Das Licht **soll ausgeschaltet werden**. |
| **dürfen** | Man darf hier nicht parken. | Hier **darf nicht geparkt werden**. |

📌 **Note:** `werden` comes at the **very end** as an Infinitive, not conjugated.

## ✅ Example Sentences with Translation

| Sentence | Translation |
|----------|-------------|
| Das Brot **wird gebacken**. | The bread is baked. |
| Der Brief **wird geschrieben**. | The letter is written. |
| Die Prüfung **wird korrigiert**. | The exam is corrected. |
| Das Auto **wird repariert**. | The car is repaired. |
| Hier **wird Deutsch gesprochen**. | German is spoken here. |
| Die Tür **muss geschlossen werden**. | The door must be closed. |
| Das Fenster **kann geöffnet werden**. | The window can be opened. |
| Im Krankenhaus **darf nicht geraucht werden**. | Smoking is not allowed in the hospital. |

## 📝 Exercises

### Exercise 1: Identify — Aktiv or Passiv?
1. Der Mann kauft ein Auto. → ___
2. Das Auto wird gekauft. → ___
3. Die Kinder spielen im Garten. → ___
4. Das Fenster wird geöffnet. → ___
5. Der Brief wird geschrieben. → ___

### Exercise 2: Convert from Aktiv to Passiv
1. Der Bäcker backt das Brot. → Das Brot ___ ___.
2. Der Mechaniker repariert das Auto. → Das Auto ___ ___.
3. Die Mutter kocht das Essen. → Das Essen ___ ___.
4. Der Schüler liest das Buch. → Das Buch ___ ___.
5. Der Künstler malt das Bild. → Das Bild ___ ___.

### Exercise 3: Fill in the correct form of werden
1. Das Wasser ___ getrunken.
2. Die Bücher ___ gelesen.
3. Der Kaffee ___ getrunken.
4. Die Blumen ___ gegossen.
5. Ich ___ gefragt.

### Exercise 4: Passiv with Modalverb
1. Man muss die Aufgabe machen. → Die Aufgabe ___ ___ ___.
2. Man kann das Problem lösen. → Das Problem ___ ___ ___.
3. Man soll den Müll trennen. → Der Müll ___ ___ ___.
4. Man darf hier nicht rauchen. → Hier ___ nicht ___ ___.
5. Man muss die Tür schließen. → Die Tür ___ ___ ___.

### Exercise 5: Correct the error
1. Das Buch wird lesen. (wrong)
2. Die Suppe wird gekocht werden. (extra word)
3. Der Brief wird von der Lehrer geschrieben. (wrong Dativ)
4. Das Fenster kann geöffnet. (missing word)
5. Die Kinder werden gefragt von der Mutter. (wrong order)

---

## ✅ Solutions

### Solution Exercise 1
1. **Aktiv** | 2. **Passiv** | 3. **Aktiv** | 4. **Passiv** | 5. **Passiv**

### Solution Exercise 2
1. Das Brot **wird gebacken**.
2. Das Auto **wird repariert**.
3. Das Essen **wird gekocht**.
4. Das Buch **wird gelesen**.
5. Das Bild **wird gemalt**.

### Solution Exercise 3
1. **wird** | 2. **werden** | 3. **wird** | 4. **werden** | 5. **werde**

### Solution Exercise 4
1. Die Aufgabe **muss gemacht werden**.
2. Das Problem **kann gelöst werden**.
3. Der Müll **soll getrennt werden**.
4. Hier **darf nicht geraucht werden**.
5. Die Tür **muss geschlossen werden**.

### Solution Exercise 5
1. Das Buch **wird gelesen**. (not lesen)
2. Die Suppe **wird gekocht**. (remove the extra werden)
3. Der Brief wird **von der Lehrerin** / **vom Lehrer** geschrieben.
4. Das Fenster kann **geöffnet werden**. (add werden)
5. Die Kinder **werden von der Mutter gefragt**. (von der Mutter before gefragt)

---

💡 **Quick tip:** The difference between Aktiv and Passiv is focus. Aktiv = the doer matters. Passiv = the action (or result) matters. In exams, the most common task is converting Aktiv to Passiv. Follow the four steps above and you'll succeed!
