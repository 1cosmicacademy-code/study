- Learn how the German present participle (Partizip I) functions as an adjective.

---

## 📚 Core Rule

The present participle (Partizip I) is formed by adding **-end** (or **-ierend** for some verbs) to the verb stem. When used as an adjective it inflects like a regular adjective and agrees with the noun in gender, number and case.

| Case | Example with **machen** | Translation |
|------|------------------------|-------------|
| Nominative Masculine | **der machende** Mann | the doing man |
| Accusative Feminine | **die machende** Frau | the doing woman |
| Dative Neuter | **dem machenden** Kind | to the doing child |
| Genitive Plural | **der machenden** Menschen | of the doing people |

| Verb | Partizip I | Example Phrase |
|------|-----------|----------------|
| **machen** | mach**end** | das **machende** Kind (the doing child)
| **sehen** | seh**end** | die **sehende** Frau (the seeing woman)
| **fahren** | fahr**end** | der **fahrende** Zug (the moving train)

### Examples

- **das lachende Kind** — the laughing child
- **der spielende Hund** — the playing dog
- **die lernende Schülerin** — the learning schoolgirl
