# قواعد كتابة المحتوى النحوي — Grammar Content Writing Rules

## 1. مستوى المحتوى
- المستوى المستهدف: **A2** حسب الإطار الأوروبي المرجعي للغات (CEFR)
- الجمهور: متعلمو اللغة الألمانية الناطقون بالعربية
- الافتراض: المتعلمون يعرفون أساسيات A1 (تقديم النفس، الأرقام، الألوان، الأفعال الأساسية)

## 2. هيكل الموضوع الإلزامي
```
1. H1 (#) — عنوان الموضوع بالعربية والألمانية
2. سطر العودة: [🔙 العودة إلى القواعد](../index.md)
3. سطر فارغ + --- + سطر فارغ
4. H2 (##) — 🎯 أهداف الدرس (قائمة نقطية)
5. سطر فارغ + --- + سطر فارغ
6. H2 (##) — 📚 القاعدة الأساسية
7. جدول يلخص القاعدة
8. H3 — أمثلة توضيحية
9. المزيد من الأقسام حسب الحاجة
10. H2 (##) — 📝 ملخص (اختياري)
```

## 3. دقة المعلومات
- **ممنوع**: تقديم معلومات نحوية غير دقيقة
- **ممنوع**: تبسيط القاعدة لدرجة فقدان المعنى
- **ممنوع**: اختراع أمثلة غير طبيعية أو غير مفهومة
- **مطلوب**: التحقق من صحة التصريفات النحوية
- **مطلوب**: ذكر الاستثناءات المهمة

## 4. اللغة والأسلوب
- الشرح الأساسي: **بالعربية**
- المصطلحات: بالإنجليزية أو الألمانية مع الترجمة
- الأمثلة: بالألمانية + الترجمة العربية
- النطق: كتابة النطق الصوتي بالأحرف العربية
- النبرة: تشجيعية، تعليمية، واضحة

## 5. تنسيق الأمثلة
```markdown
- **Ich habe ein Auto.** — أنا عندي سيارة.
- **Wir haben keine Zeit.** — نحن ليس عندنا وقت.
```

## 6. إدارة الملفات
- كل موضوع: ملفان (عربي `.md` + إنجليزي `.en.md`)
- اسم الملف: `{topic-slug}.md` و `{topic-slug}.en.md`
- مواضيع A2 فقط — لا تلمس دروس A1 الموجودة

## 7. الفيديوهات
- ابحث عن 1-3 فيديوهات مناسبة لكل موضوع
- استخدم قنوات موثوقة: Learn German, Hallo Deutschschule, Deutsch lernen mit der DW
- أضف رابط الفيديو مع شرح قصير لمحتواه
- تحقق من أن الفيديو مناسب للمستوى A2

## 8. الصيانة والتحديث
- راجع المحتوى دورياً للتأكد من دقته
- حدّث الأمثلة إن وجدت أمثلة أفضل
- استمع لملاحظات المتعلمين (في حال توفرت)

## 9. الأدوات المسموحة
- العمليات المسموحة: Read, Edit, Write, Glob, Grep
- ممنوع: حذف أو نقل ملفات موجودة
- مسموح: إنشاء ملفات جديدة ضمن `docs/grammatik/`
