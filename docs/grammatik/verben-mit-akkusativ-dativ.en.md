# Verbs with Accusative and Dative — Verben mit Akkusativ und Dativ

[🔙 Back to Grammar](../index.en.md)

---

## 🎯 Lesson Goals

- Identify verbs that require both an **Accusative** and a **Dative** object.
- Learn the correct word order: `Verb + Dative + Accusative`.
- Practice these verbs in daily contexts.

---

## 📚 Core Rule

| Verb | Dative (indirect object) + Accusative (direct object) | Example |
|------|------------------------------------------------------|---------|
| **geben** | Dativ + Akkusativ | Ich gebe **dem Mann** das Buch. — I give the man the book. |
| **schenken** | Dativ + Akkusativ | Er schenkt **der Frau** eine Blume. — He gives the woman a flower. |
| **helfen** | Dativ only | Wir helfen **dem Kind**. — We help the child. |
| **zeigen** | Dativ + Akkusativ | Du zeigst **dem Freund** das Bild. — You show the friend the picture. |
| **bringen** | Dativ + Akkusativ | Sie bringt **dem Lehrer** das Heft. — She brings the teacher the notebook. |
| **schreiben** | Dativ + Akkusativ | Ich schreibe **dem Chef** eine E‑Mail. — I write the boss an e‑mail. |

### Tips
- Some verbs need **only Dative** (e.g., *helfen*, *folgen*). No Accusative object follows.
- Verbs that need **both** always follow the order **Verb → Dative → Accusative**.

## ✅ Summary

- **Verb + Dative + Accusative** is the standard order for verbs that need both objects.
- Remember that a few verbs only require Dative.
- Practicing with everyday nouns will solidify the pattern.
