## ❌ النواقص — المهام المعلقة

### 1. 🔴 إصلاح القائمة الرئيسية (Nav i18n)

**المشكلة:** الـ `nav` في `mkdocs.yml` يستخدم عناوين عربية مباشرة. عند التبديل للإنجليزية (`/en/`)، القائمة تبقى عربية.

**الحل المطلوب:**
- تغيير `nav` إلى **مفاتيح إنجليزية** (neutral keys)
- إضافة `nav_translations` في كل لغة
- ملف واحد فقط متأثر: `mkdocs.yml`

**التفاصيل:** راجع الخطة في `C:\Users\Delta Mentor\.claude\plans\vast-imagining-mist.md`

### 2. 🔴 ربط Auto-Deploy بين GitHub و Netlify

**المشكلة:** حاليًا النشر يدوي (نفذنا `netlify deploy --prod` مرة واحدة). أي push جديد لـ `main` لا ينشر تلقائياً.

**الحل:** من Dashboard Netlify → ربط المستودع GitHub → تفعيل auto-deploy. لا يحتاج كود.

### 3. 🟡 إضافة فيديوهات إنجليزية أفضل لـ A2

بعض فيديوهات A2 الإنجليزية من قناة "Learn German" match جزئي فقط مع موضوع الدرس. يحتاج البحث عن بدائل أدق.
