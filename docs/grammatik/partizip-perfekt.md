## 💡 ما هو Partizip 2؟

**Partizip 2** (أو Partizip Perfekt) هو الصيغة الثالثة للفعل في الألمانية. يُستخدم لبناء:
- زمن **Perfekt** (الماضي التام): Ich habe **gekauft**.
- زمن **Plusquamperfekt** (الماضي الأسبق): Ich hatte **gekauft**.
- **Passiv** (المبني للمجهول): Es wird **gekauft**.
- كصفة: das **gekaufte** Auto.

---

## 📊 قواعد تشكيل Partizip 2

### 1. الأفعال المنتظمة (schwache Verben)

`ge-` + جذر الفعل + `-t`

| المصدر (Infinitiv) | الجذر (Stamm) | Partizip 2 |
|:------------------:|:-------------:|:----------:|
| machen (يفعل) | mach | **gemacht** |
| sagen (يقول) | sag | **gesagt** |
| kaufen (يشتري) | kauf | **gekauft** |
| spielen (يلعب) | spiel | **gespielt** |
| wohnen (يسكن) | wohn | **gewohnt** |
| lernen (يتعلم) | lern | **gelernt** |

### 2. الأفعال غير المنتظمة (starke Verben)

`ge-` + جذر الفعل (مع تغيير) + `-en`

| المصدر (Infinitiv) | Partizip 2 |
|:------------------:|:----------:|
| sehen (يرى) | **gesehen** |
| kommen (يأتي) | **gekommen** |
| gehen (يذهب) | **gegangen** |
| finden (يجد) | **gefunden** |
| sprechen (يتكلم) | **gesprochen** |
| schreiben (يكتب) | **geschrieben** |
| essen (يأكل) | **gegessen** |
| trinken (يشرب) | **getrunken** |
| nehmen (يأخذ) | **genommen** |
| fahren (يسوق/يذهب) | **gefahren** |

### 3. الأفعال المنتهية بـ -ieren

**بدون** ge-، فقط جذر + `-t`

| المصدر (Infinitiv) | Partizip 2 |
|:------------------:|:----------:|
| studieren (يدرس) | **studiert** |
| probieren (يجرب) | **probiert** |
| telefonieren (يتلفن) | **telefoniert** |
| korrigieren (يصحح) | **korrigiert** |
| organisieren (ينظم) | **organisiert** |
| reparieren (يصلح) | **reparariert** → **repariert** |

### 4. الأفعال المنفصلة (trennbare Verben)

`ge-` بين البادئة والفعل

| المصدر (Infinitiv) | Partizip 2 |
|:------------------:|:----------:|
| einkaufen (يشتري) | **eingekauft** |
| aufstehen (يقوم) | **aufgestanden** |
| anrufen (يتصل) | **angerufen** |
| mitbringen (يحضر معه) | **mitgebracht** |
| fernsehen (يتفرج) | **ferngesehen** |
| zumachen (يغلق) | **zugemacht** |

### 5. الأفعال غير المنفصلة (untrennbare Verben)

**بدون** ge-، فقط جذر مع بادئة

| المصدر (Infinitiv) | Partizip 2 |
|:------------------:|:----------:|
| verstehen (يفهم) | **verstanden** |
| besuchen (يزور) | **besucht** |
| bezahlen (يدفع) | **bezahlt** |
| verkaufen (يبيع) | **verkauft** |
| erklären (يشرح) | **erklärt** |
| bekommen (يحصل) | **bekommen** |

---

## 📊 جدول تصنيفي شامل

| النوع | القاعدة | مثال |
|:-----|:--------|:-----|
| منتظم (schwach) | ge- + Stamm + -t | gemacht, gesagt, gekauft |
| غير منتظم (stark) | ge- + Stamm (مغيّر) + -en | gesehen, gekommen, geschrieben |
| مختلط (gemischt) | ge- + Stamm (مغيّر) + -t | gebracht, gewusst, gedacht |
| -ieren | Stamm + -t (بدون ge-) | studiert, probiert |
| منفصل (trennbar) | Präfix + ge- + Stamm + ... | eingekauft, angerufen |
| غير منفصل (untrennbar) | Präfix + Stamm + ... | verstanden, bezahlt |

---

## ✅ أمثلة في جمل

- Ich habe gestern **eingekauft**. — أنا اشتريت أمس.
- Er ist nach Hause **gekommen**. — هو جاء إلى البيت.
- Wir haben einen Film **gesehen**. — نحن شاهدنا فيلماً.
- Hast du **verstanden**? — هل فهمت؟
- Sie hat Deutsch **studiert**. — هي درست الألمانية.
- Ich bin früh **aufgestanden**. — أنا استيقظت مبكراً.
