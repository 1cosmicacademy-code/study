## 2. 📖 شرح القاعدة — Perfekt (الماضي المساعد) بأبسط شكل

### 🧒 تخيل أنك تحكي لأبيك ماذا فعلت في حفلة أمس.

في الألمانية، لما تحكي عن شيء **خلّص وخلص** (حدث في الماضي وانتهى)، تستخدم زمن اسمه **Perfekt**.

### القاعدة ببساطة:

```
فعل مساعد (haben / sein) في المضارع + Partizip II (التصريف الثالث) في آخر الجملة
```

### 🚗 متى نستخدم haben ومتى نستخدم sein؟

| sein | haben |
|------|-------|
| ✅ حركة أو انتقال من مكان لآخر | ✅ معظم الأفعال الباقية |
| kommen (يأتي)، gehen (يذهب)، fahren (يذهب بمركبة)، reisen (يسافر) | spielen (يلعب)، tanzen (يرقص)، singen (يغني)، essen (يأكل) |
| ✅ تغير حالة: werden (يصبح)، sterben (يموت) | ✅ الأفعال التي تأخذ مفعول به (Akkusativ) |
| ✅ bleiben (يبقى)، sein (يكون) | |

> 🧠 **تذكير بسيط:**
> - **sein** = حركة أو تغير (فعل يغير مكانك أو حالتك)
> - **haben** = كل شيء آخر

### ✏️ كيف نصنع التصريف الثالث (Partizip II)؟

#### النوع الأول: الأفعال العادية (Regular / schwach)

```
**ge** + جذر الفعل + **t**
```

| المصدر | Partizip II | المعنى |
|--------|-------------|--------|
| spielen | **ge**spiel**t** | لعب |
| tanzen | **ge**tanz**t** | رقص |
| feiern | **ge**feier**t** | احتفل |
| machen | **ge**mach**t** | عمل |
| sagen | **ge**sag**t** | قال |
| kaufen | **ge**kauf**t** | اشترى |
| hören | **ge**hör**t** | سمع |
| wohnen | **ge**wohn**t** | سكن (لا تستخدم في Perfekt بكثرة) |

#### النوع الثاني: الأفعال غير العادية (Irregular / stark)

```
**ge** + جذر الفعل (غالباً يتغير) + **en**
```

| المصدر | Partizip II | المعنى |
|--------|-------------|--------|
| essen | **ge**gess**en** | أكل (e → a) |
| trinken | **ge**trunk**en** | شرب (i → u) |
| singen | **ge**sung**en** | غنى (i → u) |
| kommen | **ge**komm**en** | جاء |
| gehen | **ge**gang**en** | ذهب (يتغير الجذر كلياً) |
| helfen | **ge**holf**en** | ساعد (e → o) |
| sprechen | **ge**sproch**en** | تكلم (e → o) |
| sehen | **ge**seh**en** → gesehen | رأى |

#### 🚨 استثناءات مهمة:

1. **الأفعال المنتهية بـ -ieren:** *لا تأخذ ge-*
   - passieren → **passiert** (وليس gepassiert ❌)
   - studieren → **studiert**
   - fotografieren → **fotografiert**

2. **الأفعال ذات البادئات غير المنفصلة (be-, ver-, er-, ent-, ge-, zer-, miss-):** *لا تأخذ ge-*
   - verstehen → **verstanden**
   - besuchen → **besucht**
   - erzählen → **erzählt**

3. **الأفعال ذات البادئات المنفصلة:** *ge- بين البادئة والفعل*
   - **auf**räumen → **auf**ge**räumt**
   - **ein**laden → **ein**ge**laden**
   - **mit**bringen → **mit**ge**bracht**

---

### 📝 أمثلة بسيطة:

> Ich **habe** gestern **gefeiert**.
> أنا احتفلت أمس.

> Wir **haben** zusammen **gesungen**.
> نحن غنينا معاً.

> Er **ist** nach Hause **gekommen**.
> هو جاء إلى البيت.

> Sie **sind** zur Party **gegangen**.
> هم ذهبوا إلى الحفلة.

> Was **hast** du **mitgebracht**?
> ماذا أحضرت معك؟

> Ich **habe** Kuchen **gegessen**.
> أنا أكلت كعكة.

---

### 🎯 ملخص سريع:

```
==================== مهم جداً ====================
الجملة في Perfekt:

  haben/sein (مُصرَّف) + ... + Partizip II (في الآخر)
       ↑                                ↑
    أنا / أنت / هو ...              التصريف الثالث

مثال:
  Ich   habe   einen Kuchen   gegessen.
  أنا   عندي   كعكة           مأكولة

  (حرفياً: أنا أملك كعكة مأكولة ← معناها: أنا أكلت كعكة)
  
================================================
```

## 3. 💬 محادثة (حوار)

### 🎂 حوار 1: بعد الحفلة

| Person | Deutsch | العربية |
|--------|---------|---------|
| **Anna:** | Hallo Tom! Wie war die Party gestern? | مرحباً توم! كيف كانت الحفلة أمس؟ |
| **Tom:** | Hallo Anna! Die Party **war** super! **Hast** du **gefeiert**? | مرحباً آنا! كانت الحفلة رائعة! هل احتفلتِ؟ |
| **Anna:** | Ja, ich **habe** **gefeiert**. Ich **bin** um 8 Uhr **gekommen**. | نعم، احتفلت. جئت الساعة 8. |
| **Tom:** | Was **hast** du **mitgebracht**? | ماذا أحضرتِ معك؟ |
| **Anna:** | Ich **habe** einen Kuchen **mitgebracht**. | أحضرت كعكة معي. |
| **Tom:** | Lecker! Ich **habe** Cola und Saft **getrunken**. **Hast** du auch **getrunken**? | لذيذ! أنا شربت كولا وعصير. هل شربتِ أيضاً؟ |
| **Anna:** | Ja, ich **habe** Saft **getrunken**. **Hat** Peter **gesungen**? | نعم، شربت عصيراً. هل غنى بيتر؟ |
| **Tom:** | Ja, er **hat** drei Lieder **gesungen**! | نعم، غنى ثلاث أغانٍ! |
| **Anna:** | Super! Und Lisa? **Ist** sie **gekommen**? | رائع! وليزا؟ هل جاءت؟ |
| **Tom:** | Nein, sie **ist** nicht **gekommen**. Sie **war** krank. | لا، لم تأتِ. كانت مريضة. |
| **Anna:** | Schade! Habt ihr **getanzt**? | مؤسف! هل رقصتم؟ |
| **Tom:** | Ja, wir **haben** viel **getanzt** und **Musik gehört**. | نعم، رقصنا كثيراً واستمعنا للموسيقى. |
| **Anna:** | Und wer **hat aufgeräumt**? | ومن رتّب (نظّف)؟ |
| **Tom:** | Ich **habe aufgeräumt**. Meine Mutter **hat geholfen**. | أنا رتّبت. أمي ساعدت. |
