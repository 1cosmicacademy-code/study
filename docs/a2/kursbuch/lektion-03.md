### جمل نموذجية (Beispielsätze)

| Deutsch | Bedeutung (Arabisch) |
|---------|---------------------|
| Ich lese gern Bücher. | أحب قراءة الكتب. |
| Ich höre gern Musik. | أحب الاستماع للموسيقى. |
| Ich spiele gern Fußball. | أحب لعب كرة القدم. |
| In meiner Freizeit koche ich. | في وقت فراغي أطبخ. |
| Am Wochenende gehe ich schwimmen. | في عطلة نهاية الأسبوع أذهب للسباحة. |
| Wir treffen uns mit Freunden. | نلتقي بالأصدقاء. |
| Er sieht gern fern. | هو يحب مشاهدة التلفاز. |

> 💡 **نصيحة:** استخدم `gern` بعد الفعل للتعبير عن "أحب أن أفعل" — مثلاً `Ich lese gern` = أحب القراءة. و `nicht gern` = لا أحب.

## 3. 📝 Trennbare Verben (Wiederholung) — الأفعال المنفصلة (مراجعة)

### ما هي الأفعال المنفصلة؟

الأفعال المنفصلة (trennbare Verben) هي أفعال لها بادئة (prefix) تنفصل عن الفعل وتذهب إلى نهاية الجملة عند التصريف في المضارع.

### أشهر الأفعال المنفصلة (Wichtige trennbare Verben)

| Deutsch | Aussprache (Arabisch) | Bedeutung (Arabisch) |
|---------|----------------------|---------------------|
| fernsehen | فيرنزيين | يشاهد التلفاز |
| aufstehen | آوفشتيهن | يستيقظ |
| einkaufen | آينكاوفن | يتسوق |
| mitkommen | ميتكومن | يأتي مع |
| anfangen | آنفانگن | يبدأ |
| mitbringen | ميتبرينگن | يحضر معه |
| aufräumen | آوفرويمن | يرتب |
| zumachen | تسوماخن | يغلق |
| anrufen | آنروفن | يتصل هاتفياً |
| abholen | آبهولن | يصطحب / يأتي ليأخذ |
| vorbereiten | فوربيرايتن | يحضّر |
| einladen | آينلادن | يدعو |

### قاعدة التصريف (Konjugation)

| Personalpronomen | fernsehen | aufstehen | einkaufen | anfangen |
|-----------------|----------|----------|-----------|---------|
| ich | sehe fern | stehe auf | kaufe ein | fange an |
| du | siehst fern | stehst auf | kaufst ein | fängst an |
| er / sie / es | sieht fern | steht auf | kauft ein | fängt an |
| wir | sehen fern | stehen auf | kaufen ein | fangen an |
| ihr | seht fern | steht auf | kauft ein | fangt an |
| sie / Sie | sehen fern | stehen auf | kaufen ein | fangen an |

### أمثلة (Beispiele)

| Deutsch | Bedeutung (Arabisch) |
|---------|---------------------|
| Ich stehe um 7 Uhr auf. | أستيقظ في الساعة 7. |
| Sie sieht gern fern. | هي تحب مشاهدة التلفاز. |
| Wir kaufen am Samstag ein. | نحن نتسوق يوم السبت. |
| Wann fängt der Kurs an? | متى تبدأ الدورة؟ |
| Ich lade dich zur Party ein. | أدعوك إلى الحفلة. |

> ⚠️ **قاعدة ذهبية:** البادئة المنفصلة تذهب إلى **نهاية الجملة** دائماً! مثلاً: `Ich mache das Fenster zu` (أغلق النافذة) — وليس `Ich zumache`.

### في الجمل الثانوية (Nebensätze)

في الجمل الثانوية التي تبدأ بـ `dass, weil, wenn`، البادئة لا تنفصل:

| Deutsch | Bedeutung (Arabisch) |
|---------|---------------------|
| ... weil ich früh aufstehe. | ... لأنني أستيقظ مبكراً. |
| ... dass sie gern fernsieht. | ... أنها تحب مشاهدة التلفاز. |

## 4. 💬 Dialog: "In unserer Freizeit" — حوار: في وقت فراغنا

**Personen:** Tom und Lisa (Freunde)

| Deutsch | Bedeutung (Arabisch) |
|---------|---------------------|
| **Tom:** Hallo Lisa! Was machst du heute Nachmittag? | مرحباً ليزا! ماذا تفعلين بعد الظهر؟ |
| **Lisa:** Ich habe frei. Ich möchte schwimmen gehen. | أنا عطلة. أرغب في الذهاب للسباحة. |
| **Tom:** Das klingt gut! Ich gehe auch gern schwimmen. | هذا يبدو جيداً! أنا أيضاً أحب السباحة. |
| **Lisa:** Super! Kommst du mit? | رائع! هل ستأتي معي؟ |
| **Tom:** Ja, gern! Aber ich muss zuerst einkaufen. | نعم، بكل سرور! لكن يجب أن أتسوق أولاً. |
| **Lisa:** Kein Problem. Ich warte auf dich. | لا مشكلة. أنا أنتظرك. |
| **Tom:** Um wie viel Uhr treffen wir uns? | في أي ساعة نلتقي؟ |
| **Lisa:** Um 15 Uhr am Schwimmbad. Um 15 Uhr fängt der Kurs an. | الساعة 3 في المسبح. الساعة 3 تبدأ الحصة. |
| **Tom:** Perfekt! Ich bringe meine Sachen mit. | ممتاز! سأحضر أغراضي معي. |
| **Lisa:** Toll! Bis später! | رائع! إلى اللقاء! |

> 💡 **تعبير مفيد:** "Das klingt gut!" = هذا يبدو جيداً. "Kein Problem!" = لا مشكلة. "Bis später!" = إلى اللقاء.
