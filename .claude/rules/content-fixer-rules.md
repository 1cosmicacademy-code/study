# Content Fixer — قواعد العمل

## القواعد الأساسية

1. **YAML أولاً** — قبل تعديل أي شيء آخر، تأكد من صحة YAML syntax. خطأ YAML واحد يكسر البناء كله.
2. **اختبر بعد كل تعديل** — بعد أي تغيير في `mkdocs.yml`، شغّل `mkdocs build --verbose` فوراً.
3. **لا تؤثر على العربية** — عند إصلاح الصفحات الإنجليزية، تأكد من أن الصفحات العربية لم تتأثر.
4. **أقل تعديل** — لا تغير كل الإعدادات، أصلح المشكلة فقط.
5. **لا تحذف ملفات** — لا تحذف ملفات موجودة إلا بتعليمات صريحة.

## Must / Must Not

- **يجب (Must):**
  - قراءة `mkdocs.yml` كاملاً قبل أي تعديل
  - عمل `mkdocs build --verbose` بعد كل تعديل
  - فحص 3 صفحات عربية و 3 صفحات إنجليزية بعد إصلاح RTL
  - التحقق من أن favicon.ico يظهر في تبويب المتصفح
  - كتابة robots.txt بصيغة صحيحة
  - استخدام مسارات مطلقة في التعديلات

- **ممنوع (Must Not):**
  - تعديل محتوى الدروس أو التمارين النصية
  - تعديل ملفات JS (إلا إذا كان الإصلاح في كيفية تحميلها)
  - حذف صفحات موجودة
  - تجاهل تحذيرات `mkdocs build` (WARNING قد تخفي مشاكل أكبر)
  - افتراض أن المسار صحيح دون فتح الصفحة

## معايير الجودة

### 1. دقة التعديل (Edit Accuracy)
- تغيير واحد في YAML لا يكسر YAML syntax
- **مؤشر الأداء:** 100% من التعديلات تبني بنجاح

### 2. شمولية الإصلاح (Fix Completeness)
- المشكلة لا تعود بعد الإصلاح
- **مؤشر الأداء:** 0 regression

### 3. سرعة التنفيذ (Execution Speed)
- إصلاح YAML بسيط: < 5 دقائق
- إنشاء صفحة جديدة: < 15 دقيقة
- **مؤشر الأداء:** الالتزام بالأوقات

### 4. التوثيق (Documentation)
- سجل التغييرات التي أجريتها
- اشرح لماذا تم هذا التعديل
- **مؤشر الأداء:** 100% من التعديلات موثقة

## مشاكل شائعة في YAML

```
# خطأ: dir: 'rtl' (عام على كل الصفحات)
# الصواب: dir: rtl (مشروط حسب اللغة)

# خطأ: missing space بعد :
key:value  ← خطأ
key: value  ← صحيح

# مسار خاطئ
- Grammatik/  ← إذا كان المجلد صغير G لكن الملف يبدأ big G
```

## قائمة التحقق قبل التسليم

- [ ] هل `mkdocs build --verbose` يمر بدون ERROR؟
- [ ] هل الصفحات العربية لا تزال RTL؟
- [ ] هل الصفحات الإنجليزية أصبحت LTR (إن كان هذا هو الإصلاح)؟
- [ ] هل الرابط المُصلَح يعطي 200؟
- [ ] هل JS يُحمّل فقط في الصفحات التي تحتاجه؟
- [ ] هل favicon.ico و robots.txt موجودان ويُخدمان؟
- [ ] هل التمارين الإنجليزية تظهر بالإنجليزية فقط؟
