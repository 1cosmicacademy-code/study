> 🎯 **اكتشف مستواك في اللغة الألمانية** — اختبار تشخيصي من 42 سؤالاً يحدد مستواك التقريبي (A1/A2/B1) ويكشف نقاط قوتك وضعفك مع توصيات مخصصة.

## 🎯 لماذا هذا الاختبار؟

| السؤال | الجواب |
|--------|--------|
| **ما مستواي الحالي؟** | A1 (مبتدئ) / A2 (متوسط) / B1 (متوسط متقدم) |
| **ما هي نقاط قوتي؟** | المجالات التي أجبت فيها correctly |
| **ما هي نقاط ضعفي؟** | القواعد والدروس التي تحتاج مراجعة |
| **من أين أبدأ؟** | توصية دقيقة بمستوى ودرس محدد |
| **ما القواعد التي أحتاجها؟** | روابط مباشرة لكل قاعدة نحوية |
| **أي تمارين تحلّها؟** | روابط لتمارين مناسبة لمستواك |

## 📋 تعليمات

1. **أجب عن جميع الأسئلة** — اختر الإجابة الصحيحة أو اكتبها
2. **لا تستخدم قاموساً** — هذا اختبار تشخيصي لقياس معرفتك الحالية
3. **لا توجد إجابة "خاطئة"** — الاختبار يكشف نقاط القوة والضعف فقط
4. **سيظهر تقرير كامل** بنقاط القوة والضعف والتوصيات بعد الضغط على **بدء التقييم**
