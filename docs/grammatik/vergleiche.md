# المقارنة (Vergleiche) في اللغة الألمانية

🎯 **المقارنة** هي طريقة نقارن بها بين شيئين أو أكثر. في الألمانية، نستخدم **wie** للمساواة و **als** للتفاضل. بالإضافة إلى ذلك، الصفات تأخذ أشكالاً خاصة في المقارنة والتفضيل.

---

## 💡 متى نستخدم wie ومتى نستخدم als؟

### 1. wie — مثل / بنفس المستوى (مساواة)

نستخدم **wie** عندما يكون الشيئين متساويين أو متشابهين.

```
Er ist so groß wie ich.
هو طويل مثلي.
```

```
Sie singt so schön wie ihre Mutter.
هي تغني بشكل جميل مثل أمها.
```

> ⚠️ **انتبه:** قبل **wie** غالباً تأتي كلمة **so** أو **genauso** (نفس الشيء).

### 2. als — من (تفاضل)

نستخدم **als** عندما يكون هناك فرق بين الشيئين وأحدهما أكثر أو أقل.

```
Er ist größer als ich.
هو أطول مني.
```

```
Deutsch ist schwerer als Englisch.
الألمانية أصعب من الإنجليزية.
```

## ⚠️ القاعدة: تصريف الصفات للمقارنة

### 🟢 الصفة العادية (Positiv)

الصفة بشكلها الطبيعي:
- **groß** — كبير / طويل
- **klein** — صغير
- **schnell** — سريع
- **schön** — جميل
- **teuer** — غالي
- **gut** — جيد
- **gern** — بكل سرور

### 🟢 المقارنة (Komparativ)

الصفة + **-er** (بمعنى "أكثر... من"):

| الصفة الأصلية | المقارنة (Komparativ) | المعنى |
|:-------------:|:---------------------:|:------:|
| **groß** | **größer** (größ + er) | أكبر / أطول |
| **klein** | **kleiner** | أصغر |
| **schnell** | **schneller** | أسرع |
| **schön** | **schöner** | أجمل |
| **teuer** | **teurer** | أغلى |
| **billig** | **billiger** | أرخص |
| **langsam** | **langsamer** | أبطأ |

> ⚠️ ملاحظات مهمة:
> - الكلمات ذات حرف العلة a, o, u غالباً تأخذ **Umlaut** (ä, ö, ü): groß → **größer**
> - الكلمات التي تنتهي بـ **er** تحذف الـ e: teuer → **teurer**
> - الكلمات التي تنتهي بـ **el** تحذف الـ e: dunkel → **dunkler**

### 🟢 التفضيل (Superlativ)

طريقتان:

1. **am + الصفة + -(e)sten** (عندما لا تذكر الاسم):
   - **am größten** — الأكبر / الأطول
   - **am schnellsten** — الأسرع

2. **der/die/das + الصفة + -(e)ste** (عندما تذكر الاسم):
   - **der größte Mann** — أطول رجل
   - **die schnellste Frau** — أسرع امرأة

| الصفة الأصلية | Komparativ | Superlativ (am ...) |
|:-------------:|:----------:|:------------------:|
| **groß** | **größer** | **am größten** |
| **schnell** | **schneller** | **am schnellsten** |
| **schön** | **schöner** | **am schönsten** |
| **jung** | **jünger** | **am jüngsten** |
| **alt** | **älter** | **am ältesten** |

### 🟢 حالات خاصة (غير منتظمة)

| الصفة الأصلية | Komparativ | Superlativ |
|:-------------:|:----------:|:----------:|
| **gut** (جيد) | **besser** | **am besten** |
| **viel** (كثير) | **mehr** | **am meisten** |
| **gern** (بكل سرور) | **lieber** | **am liebsten** |
| **hoch** (عالٍ) | **höher** | **am höchsten** |
| **nah** (قريب) | **näher** | **am nächsten** |

## ✅ جمل أمثلة مع الترجمة

| الجملة | الترجمة |
|--------|---------|
| Ich bin so alt wie du. | أنا في نفس عمرك. |
| Er läuft schneller als sein Bruder. | هو يركض أسرع من أخيه. |
| Wien ist schöner als Berlin? | فيينا أجمل من برلين؟ |
| Das Auto ist teurer als das Fahrrad. | السيارة أغلى من الدراجة. |
| Deutsch ist am schwersten. | الألمانية الأصعب. |
| Der Mount Everest ist der höchste Berg der Welt. | إيفرست أعلى جبل في العالم. |
| Ich esse am liebsten Pizza. | أحب أكل البيتزا أكثر شيء. |
| Sie ist die beste Schülerin in der Klasse. | هي أفضل طالبة في الصف. |

---

💡 **نصيحة:** لتتذكر الفرق بين wie و als: **wie = مساواة (so ... wie)** و **als = تفاضل (...er als)**. كلمة "wie" فيها حرف i مثل "في نفس المستوى"، و "als" تشبه "else" = غير ذلك (أكثر من).
