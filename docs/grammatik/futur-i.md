# زمن المستقبل (Futur I) في اللغة الألمانية

🎯 **Futur I** هو زمن المستقبل في الألمانية. نستخدمه للتعبير عن نوايا، توقعات، وعود، وتخمينات. يتكون من الفعل المساعد **werden** + الفعل الأساسي في المصدر (Infinitiv) في آخر الجملة.

---

## 💡 متى نستخدم Futur I؟

### 1. النوايا والخطط (Absichten / Pläne)

عندما نخطط لعمل شيء في المستقبل:

```
Ich werde morgen einkaufen gehen.
سأذهب للتسوق غداً.
```

```
Wir werden im Sommer nach Deutschland reisen.
سنسافر إلى ألمانيا في الصيف.
```

### 2. التوقعات (Vermutungen)

عندما نتوقع حدوث شيء:

```
Es wird morgen regnen.
سوف تمطر غداً.
```

```
Der Film wird bestimmt gut sein.
الفيلم سيكون جيداً بالتأكيد.
```

### 3. الوعود (Versprechen)

عندما نعد بشيء:

```
Ich werde dich anrufen.
سأتصل بك.
```

```
Ich werde dir helfen.
سأساعدك.
```

### 4. التخمينات (Vermutungen في الوقت الحاضر)

يمكن أن يعبر Futur I عن تخمين في الوقت الحاضر (معنى "ربما" أو "من المحتمل"):

```
Er wird jetzt im Büro sein.
ربما هو في المكتب الآن.
```

```
Sie wird wohl keine Zeit haben.
ربما ليس عندها وقت.
```

## ⚠️ القاعدة: تصريف werden وتركيب الجملة

### 🟢 تصريف werden (كفعل مساعد)

| الضمير | werden | النطق |
|:------:|:------:|:-----:|
| ich | **werde** | فيرده |
| du | **wirst** | فيرست |
| er / sie / es | **wird** | فيرد |
| wir | **werden** | فيردن |
| ihr | **werdet** | فيردت |
| sie | **werden** | فيردن |
| Sie | **werden** | فيردن |

### 🟢 تركيب الجملة

```
werden (متصرف في المكان الثاني) + ... + Infinitiv (آخر الجملة)

Ich    werde    morgen    nach Berlin    fahren.
أنا    سـ       غداً       إلى برلين     أسافر.
```

### 🟢 الفرق بين Präsens (للمستقبل) و Futur I

في الألمانية، غالباً نستخدم **Präsens** مع ظرف زمان للمستقبل بدلاً من Futur I:

| Präsens للمستقبل (أكثر شيوعاً) | Futur I (أكثر رسمية / تأكيداً) |
|:-----------------------------:|:------------------------------:|
| **Ich fliege morgen nach Berlin.** | **Ich werde morgen nach Berlin fliegen.** |
| (أنا أسافر غداً إلى برلين) | (سأسافر غداً إلى برلين) |
| **Nächstes Jahr kaufe ich ein Auto.** | **Nächstes Jahr werde ich ein Auto kaufen.** |
| (السنة القادمة أشتري سيارة) | (السنة القادمة سأشتري سيارة) |

> 💡 **ملاحظة:** في المحادثات اليومية، الألمان يستخدمون Präsens بكثرة للمستقبل. Futur I يستخدم أكثر عندما تريد **التأكيد** على النية أو عندما **تتوقع** شيئاً.

## ✅ جمل أمثلة مع الترجمة

| الجملة | الترجمة |
|--------|---------|
| Ich werde morgen früher aufstehen. | سأستيقظ مبكراً غداً. |
| Wirst du mich besuchen? | هل ستزورني؟ |
| Er wird nächstes Jahr nach Deutschland fliegen. | سيسافر إلى ألمانيا السنة القادمة. |
| Wir werden pünktlich sein. | سنكون في الموعد. |
| Ihr werdet das Examen bestimmt bestehen. | أنتم ستنجحون في الامتحان بالتأكيد. |
| Es wird heute Abend nicht regnen. | لن تمطر هذا المساء. |
| Sie wird wohl im Urlaub sein. | ربما هي في عطلة. |
| Ich werde dich nie vergessen. | لن أنساك أبداً. |

## ✅ الحلول

### حل تمرين 1
1. werde | 2. wirst | 3. wird | 4. werden | 5. werdet | 6. werden

### حل تمرين 2
1. Ich werde ein Haus kaufen.
2. Sie wird nach Italien reisen.
3. Wir werden Deutsch lernen.
4. Der Zug wird um 10 Uhr kommen.
5. Wirst du das Fenster zumachen?

### حل تمرين 3
1. Morgen werde ich mein Zimmer aufräumen.
2. Nächstes Jahr werden wir ein Haus bauen.
3. Wirst du mich am Wochenende besuchen?
4. Es wird heute nicht schneien.
5. Wir werden die Prüfung bestimmt bestehen.

### حل تمرين 4
1. kaufe (Präsens هو الأكثر شيوعاً في المحادثة)
2. wird kommen (توقع قوي ← Futur I)
3. werde ich beginnen (نية مؤكدة ← Futur I)
4. wird regnen (توقع الطقس ← Futur I)
5. werden besuchen (وعد ← Futur I)

---

💡 **نصيحة:** في البداية، استخدم Präsens + ظرف زمان للمستقبل (مثل morgen, nächstes Jahr). هذا سهل جداً ويستخدم يومياً. استخدم Futur I عندما تريد التأكيد على النية، أو التوقع، أو الوعد. مثلاً: **"Ich rufe dich morgen an."** (Präsens) vs. **"Ich werde dich anrufen, versprochen!"** (Futur I + وعد).
