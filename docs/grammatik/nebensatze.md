# الجمل الثانوية (Nebensätze) في اللغة الألمانية

🎯 **الجمل الثانوية (Nebensätze)** هي جمل تعتمد على جملة رئيسية (Hauptsatz) وترتبط بها باستخدام أدوات ربط مثل dass, weil, wenn, obwohl. الفرق الأهم: في الجملة الثانوية، الفعل يروح **آخر الجملة**!

---

## 💡 متى نستخدم الجمل الثانوية؟

### 1. dass — أنَّ (للحقائق والتصريحات)

نستخدم **dass** عندما ننقل حقيقة أو معلومة أو رأي.

```
Ich weiß, dass er kommt.
أنا أعلم أنه يأتي.
```

```
Er sagt, dass er krank ist.
هو يقول إنه مريض.
```

### 2. weil — لأن (للتعبير عن السبب)

نستخدم **weil** لإعطاء سبب.

```
Ich lerne Deutsch, weil es Spaß macht.
أنا أتعلم الألمانية لأنها ممتعة.
```

```
Er bleibt zu Hause, weil er krank ist.
هو يبقى في البيت لأنه مريض.
```

### 3. wenn — إذا / عندما (للشرط أو الوقت)

نستخدم **wenn** للشرط (إذا) أو عندما يتكرر شيء.

```
Wenn ich Zeit habe, gehe ich spazieren.
إذا كان عندي وقت، أتمشى.
```

```
Ich freue mich, wenn du kommst.
أنا أفرح عندما تأتي.
```

### 4. obwohl — على الرغم من (للتعارض / التضاد)

نستخدم **obwohl** عندما تكون المعلومة في الجملة الثانية عكس المتوقع.

```
Obwohl es regnet, gehe ich spazieren.
على الرغم من أن المطر يهطل، أتمشى.
```

```
Er kommt, obwohl er krank ist.
هو يأتي على الرغم من أنه مريض.
```

---

## ⚠️ القاعدة: ترتيب الكلمات في الجملة الثانوية

| الجملة الرئيسية | أداة الربط | الجملة الثانوية |
|:--------------:|:----------:|:--------------:|
| Ich weiß, | **dass** | er **kommt**. |
| Ich lerne Deutsch, | **weil** | es Spaß **macht**. |
| Ich freue mich, | **wenn** | du **kommst**. |
| Er kommt, | **obwohl** | er krank **ist**. |

### القاعدة الذهبية:
```
في الجملة الثانوية (Nebensatz) ← الفعل يتأخر لآخر الجملة!
```

### الفرق بين Hauptsatz و Nebensatz:

| الجملة الرئيسية (Hauptsatz) | الجملة الثانوية (Nebensatz) |
|:--------------------------:|:--------------------------:|
| فعل في **المكان الثاني** | فعل في **آخر الجملة** |
| تستطيع الوقوف وحدها | **لا تستطيع** الوقوف وحدها |
| **Ich komme morgen.** | **..., weil ich morgen komme.** |

## ✅ جمل أمثلة مع الترجمة

| الجملة | الترجمة |
|--------|---------|
| Ich denke, dass er Recht hat. | أعتقد أنه على حق. |
| Sie sagt, dass sie müde ist. | هي تقول إنها متعبة. |
| Wir bleiben zu Hause, weil es kalt ist. | نبقى في البيت لأن الجو بارد. |
| Er ruft an, weil er Hilfe braucht. | هو يتصل لأنه يحتاج مساعدة. |
| Wenn ich Hunger habe, esse ich etwas. | عندما أشعر بالجوع، آكل شيئاً. |
| Wenn du willst, können wir gehen. | إذا أردت، نستطيع الذهاب. |
| Obwohl er wenig Zeit hat, hilft er mir. | على الرغم من أن وقته قليل، هو يساعدني. |

## 📝 تمارين

### تمرين 1: اختر أداة الربط الصحيحة (dass, weil, wenn, obwohl)

1. Ich glaube, _______ er bald kommt.
2. Sie ist glücklich, _______ sie Urlaub hat.
3. _______ es regnet, gehen wir spazieren.
4. _______ ich Zeit habe, lese ich ein Buch.
5. Er sagt, _______ er kein Geld hat.

### تمرين 2: أكمل الجملة بوضع الفعل في المكان الصحيح

1. Ich hoffe, dass du morgen ___________. (kommen)
2. Er ist traurig, weil seine Katze ____________. (sterben)
3. __________ du müde bist, kannst du schlafen. (wenn)
4. Sie geht zur Arbeit, obwohl sie Fieber ____________. (haben)
5. Wir bleiben zu Hause, weil es ____________. (regnen)

### تمرين 3: ربط الجمل — حول الجملتين إلى جملة واحدة باستخدام weil

1. Ich bin müde. Ich habe viel gearbeitet.
   → Ich bin müde, _______________.

2. Er kommt nicht. Er ist krank.
   → Er kommt nicht, _______________.

3. Sie lernt Deutsch. Sie will in Berlin studieren.
   → Sie lernt Deutsch, _______________.

### تمرين 4: رتب الكلمات لتكوين جملة صحيحة 🔀

1. dass / Ich / er / denke / kommt
   → ___________________________________

2. weil / lernt / Sie / Deutsch / sie / liebt / Sprache / die
   → ___________________________________

3. wenn / rufe / du / an / ich / mich / brauchst
   → ___________________________________

4. obwohl / geht / ins / er / er / Kino / krank / ist
   → ___________________________________

### تمرين 5: أكمل الجملة بجملة ثانوية مناسبة

1. Ich bin froh, dass ___________________.
2. Er bleibt zu Hause, weil ___________________.
3. ___________________, wenn ich Zeit habe.
4. ___________________, obwohl es schwierig ist.

## ✅ الحلول

### حل تمرين 1
1. dass | 2. weil | 3. Obwohl | 4. Wenn | 5. dass

1. Ich bin müde, weil ich viel gearbeitet habe.
2. Er kommt nicht, weil er krank ist.
3. Sie lernt Deutsch, weil sie in Berlin studieren will.
