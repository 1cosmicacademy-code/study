## 🎯 Lesson Objectives

- Understand when to use the "um...zu" infinitive clause.
- Understand when to use the "damit" subordinate clause.
- Build sentences that express purpose clearly.

---

## 📚 Core Rule

**1. "um...zu"** creates a purpose clause with an infinitive. It links a main clause (the reason) to an action that should happen in order to achieve that reason. The infinitive is always preceded by **zu**.

```
[Main clause] + um + [infinitive with zu]
```

**2. "damit"** introduces a purpose clause that contains its own finite verb. It is used when the purpose clause needs its own subject or a full verb phrase.

```
[Main clause] + damit + [verb in subordinate clause]
```

---

### Examples

- **um...zu**
  - Ich lerne Deutsch, **um** in Berlin **zu** arbeiten.
- **damit**
  - Ich lerne Deutsch, **damit** ich in Berlin arbeiten kann.
