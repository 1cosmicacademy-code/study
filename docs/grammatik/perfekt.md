# زمن الماضي التام (Perfekt) في اللغة الألمانية

🎯 **Perfekt** هو الزمن الماضي الأكثر استخداماً في **الكلام اليومي والمحادثة** في الألمانية. إذا كنت تتكلم عن شيء صار في الماضي، فغالباً بتستخدم Perfekt.

---

## 💡 متى نستخدم Perfekt؟

| الاستخدام | مثال |
|-----------|------|
| **الحديث عن أحداث الماضي** | Gestern **habe** ich **gearbeitet**. |
| **في المحادثات اليومية** | Was **hast** du **gemacht**? |
| **أحداث اكتملت وانتهت** | Ich **bin** nach Berlin **gefahren**. |

⚠️ **الفرق:** في الكتابة والسرد (القصص والروايات) نستخدم Prateritum أكثر.  
Perfekt هو زمن **المحادثة**.

---

## ⚠️ تكوين Perfekt: haben/sein + Partizip II

```
           haben / sein  (مصرف)  +  Partizip II  (في آخر الجملة)
                |                         |
            (في المكان الثاني)        (في آخر الجملة)
```

**مثال:**
```
Ich habe einen Kuchen gebacken.
أنا   عندي    كعكة      خبزت
أنا خبزت كعكة.
```

```
Sie ist nach Hause gegangen.
هي   ذهبت   إلى   البيت
هي ذهبت إلى البيت.
```

---

## 🎯 متى نستخدم haben ومتى نستخدم sein؟

### ✅ نستخدم **sein** مع:

| الحالة | مثال |
|--------|------|
| **أفعال الحركة (تغيير مكان)** | gehen, fahren, kommen, reisen, laufen |
| **تغيير حالة** | aufwachen, einschlafen, sterben, werden |
| **أفعال البقاء** | bleiben, sein |

**أفعال sein الشائعة:**
- gehen → **ist gegangen** (ذهب)
- kommen → **ist gekommen** (جاء)
- fahren → **ist gefahren** (ذهب بمركبة)
- fliegen → **ist geflogen** (طار)
- schwimmen → **ist geschwommen** (سبح)
- reisen → **ist gereist** (سافر)
- aufwachen → **ist aufgewacht** (استيقظ)
- einschlafen → **ist eingeschlafen** (غلبته النعاس)
- bleiben → **ist geblieben** (بقي)
- sein → **ist gewesen** (كان)
- passieren → **ist passiert** (حدث)

### ✅ نستخدم **haben** مع باقي الأفعال:

| الحالة | مثال |
|--------|------|
| **أفعال مع مفعول به (Akkusativ)** | Ich **habe** das Buch **gelesen**. |
| **الأفعال الانعكاسية** | Ich **habe** mich **gewaschen**. |
| **أفعال الحالة** | Ich **habe** geschlafen. |
| **أفعال مساعدة** | Ich **habe** das **gewollt**. |

## ⚠️ تكوين Partizip II

### 1. الأفعال المنتظمة (Schwache Verben)

```
ge- + جذر الفعل + -t
```

| المصدر | Partizip II |
|--------|-------------|
| machen | **ge-mach-t** |
| kaufen | **ge-kauf-t** |
| sagen | **ge-sag-t** |
| spielen | **ge-spiel-t** |
| wohnen | **ge-wohn-t** |

⚠️ **الأفعال اللي تنتهي بـ -ieren: (بدون ge-)**
- studieren → **studiert**
- telefonieren → **telefoniert**
- fotografieren → **fotografiert**

⚠️ **الأفعال اللي تبدأ بـ be-, ver-, ent-, er-, zer-, ge-: (بدون ge-)**
- besuchen → **besucht**
- verstehen → **verstanden** (شاذة لكن قاعدة)
- erzahlen → **erzahlt**

### 2. الأفعال غير المنتظمة (Starke Verben)

```
ge- + جذر الفعل (معدل) + -en
```

| المصدر | Partizip II |
|--------|-------------|
| kommen | **ge-komm-en** |
| gehen | **ge-gang-en** |
| sehen | **ge-seh-en** |
| schreiben | **ge-schrieb-en** |

### 3. الأفعال المنفصلة (Trennbare Verben)

```
الجزء المنفصل + ge + جذر الفعل + t/en
```

| المصدر | Partizip II |
|--------|-------------|
| einkaufen | **ein-ge-kauf-t** |
| aufstehen | **auf-ge-stand-en** |
| anfangen | **an-ge-fang-en** |

---

## 📚 الأفعال الشاذة (Starke Verben) الأكثر شيوعاً

| المصدر | المعنى | Partizip II |
|--------|--------|-------------|
| sein | يكون | **ist gewesen** |
| haben | يملك | **hat gehabt** |
| werden | يصبح | **ist geworden** |
| gehen | يذهب | **ist gegangen** |
| kommen | يأتي | **ist gekommen** |
| fahren | يقود/يذهب | **ist gefahren** |
| essen | يأكل | **hat gegessen** |
| trinken | يشرب | **hat getrunken** |
| sehen | يرى | **hat gesehen** |
| lesen | يقرأ | **hat gelesen** |
| schreiben | يكتب | **hat geschrieben** |
| sprechen | يتكلم | **hat gesprochen** |
| finden | يجد | **hat gefunden** |
| geben | يعطي | **hat gegeben** |
| nehmen | يأخذ | **hat genommen** |

## ✅ جمل أمثلة مع الترجمة

| الجملة | الترجمة |
|--------|---------|
| Ich **habe** gestern einen Film **gesehen**. | رأيت فيلماً أمس. |
| Wir **sind** nach Berlin **gefahren**. | ذهبنا إلى برلين. |
| Er **hat** Deutsch **gelernt**. | هو تعلم الألمانية. |
| Sie **ist** um 8 Uhr **aufgestanden**. | هي استيقظت الساعة 8. |
| Ich **habe** mein Handy **vergessen**. | نسيت جوالي. |
| Was **hast** du **gegessen**? | ماذا أكلت؟ |
| **Bist** du schon **angekommen**? | هل وصلت بالفعل؟ |
