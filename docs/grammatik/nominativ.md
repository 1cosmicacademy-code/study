# Nominativ — حالة الرفع في اللغة الألمانية

🎯 **Nominativ** هي الحالة الأساسية في الألمانية. هي الحالة الطبيعية للفاعل (Subject) — اللي بيعمل الفعل.

---

## 💡 متى نستخدم Nominativ؟

### 1. الفاعل (Subject) — اللي بيسأل "مَن؟" أو "ماذا؟"

اللي بيعمل الفعل في الجملة.

```
Der Mann schläft. → Wer schläft? → Der Mann.
الرجل ينام.
```

```
Das Kind spielt. → Was spielt? → Das Kind.
الطفل يلعب.
```

### 2. بعد الفعل "sein" (يكون)

```
Ich bin ein Student. → أنا طالب.
Das ist ein Tisch. → هذا طاولة.
```

### 3. بعد الفعل "werden" (يصير)

```
Er wird ein Lehrer. → هو يصبح مدرساً.
```

### 4. في التحيات والعبارات الثابتة

```
Guten Morgen! → صباح الخير!
(Das ist) Schade! → هذا مؤسف!
```

## ⚠️ أدوات التعريف في Nominativ

| الأداة | مذكر (m) | مؤنث (f) | محايد (n) | جمع (pl) |
|-------|---------|---------|-----------|---------|
| **المعرفة (the)** | **der** | **die** | **das** | **die** |
| **النكرة (a/an)** | **ein** | **eine** | **ein** | --- |
| **النفي (no)** | **kein** | **keine** | **kein** | **keine** |

**أمثلة:**
- **Der** Mann ist alt. — الرجل عجوز.
- **Die** Frau ist jung. — المرأة شابة.
- **Das** Kind ist klein. — الطفل صغير.
- **Ein** Hund bellt. — كلب ينبح.
- **Keine** Zeit. — لا وقت.

## 🎯 قاعدة ذهبية

```
Nominativ = الفاعل (subject) = اللي بيعمل الفعل
أداة المذكر = der, أداة المؤنث = die, أداة المحايد = das
```

## ✅ جمل أمثلة

| الجملة | الترجمة |
|--------|---------|
| **Der** Hund ist braun. | الكلب بني. |
| **Die** Katze ist schwarz. | القطة سوداء. |
| **Das** Haus ist groß. | المنزل كبير. |
| **Die** Kinder spielen. | الأطفال يلعبون. |
| **Ein** Mann kommt. | رجل يأتي. |
| **Eine** Frau singt. | امرأة تغني. |
| **Kein** Problem! | لا مشكلة! |

## 📝 تمارين

### تمرين 1: اختر الأداة الصحيحة (der/die/das)

1. ___ Tisch (طاولة)
2. ___ Lampe (مصباح)
3. ___ Buch (كتاب)
4. ___ Stühle (كراسي)
5. ___ Fenster (نافذة)

### تمرين 2: ein/eine/kein/keine

1. Das ist ___ Hund. (ein)
2. Das ist ___ Katze. (eine)
3. Das ist ___ Problem. (kein — ليس مشكلة)
4. Das sind ___ Blumen. (keine — ليست زهور)
5. Das ist ___ Auto. (ein)

### تمرين 3: أكمل الجملة

1. ___ Sonne scheint. (الشمس — die)
2. ___ Himmel ist blau. (السماء — der)
3. ___ Kinder sind laut. (الأطفال — die)
4. ___ Apfel ist rot. (تفاحة — der)
5. ___ Wasser ist kalt. (الماء — das)

## ✅ الحلول

### حل تمرين 1
1. der | 2. die | 3. das | 4. die | 5. das

### حل تمرين 2
1. ein | 2. eine | 3. kein | 4. keine | 5. ein

### حل تمرين 3
1. Die | 2. Der | 3. Die | 4. Der | 5. Das
