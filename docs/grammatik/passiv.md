# المبني للمجهول (Passiv) في اللغة الألمانية

🎯 **Passiv** (المبني للمجهول) نستخدمه لما **الفعل هو المهم** مش الفاعل. يعني "الباب اتنفتح" (مش مهم مين فتحه) بدل "أحمد فتح الباب".

---

## 💡 الفرق بين Aktiv und Passiv

| Aktiv (معلوم) | Passiv (مجهول) |
|---------------|----------------|
| الفاعل هو اللي بيعمل الفعل | الفعل بيحصل — الفاعل مش مهم أو مش معروف |
| **Der Koch** kocht die Suppe. | Die Suppe **wird gekocht**. |
| الطباخ يطبخ الشوربة. | الشوربة **تُطبخ** (مش مهم مين). |

---

## ⚠️ تكوين Passiv

### القاعدة الأساسية:
> **werden (مصرف) + Partizip II (في نهاية الجملة)**

```
[Subject] + werden (conjugated) + ... + Partizip II
```

**أمثلة:**
- `Das Essen **wird gekocht**.` — الطعام يُطبخ.
- `Die Tür **wird geöffnet**.` — الباب يُفتح.
- `Die Hausaufgaben **werden gemacht**.` — الواجبات تؤدى.

### تصريف werden في المضارع مع Passiv:

| الضمير | werden + Partizip II |
|--------|----------------------|
| **ich** | werde gefragt |
| **du** | wirst gefragt |
| **er/sie/es** | wird gefragt |
| **wir** | werden gefragt |
| **ihr** | werdet gefragt |
| **sie/Sie** | werden gefragt |

---

## 🎯 خطوات تحويل Aktiv → Passiv

هذه **خطة بسيطة** للتحويل:

| الخطوة | Aktiv | Passiv |
|--------|-------|--------|
| 1️⃣ | **Der Koch** kocht die Suppe. | ← ابحث عن الفاعل (Der Koch) |
| 2️⃣ | Der Koch kocht **die Suppe**. | → **Die Suppe** تصير Subject (Nominativ) |
| 3️⃣ | Der Koch kocht die Suppe. | → الفعل kocht يصير **wird gekocht** |
| 4️⃣ | ✂️ Der Koch | → الفاعل القديم إما يحذف أو يصير **von + Dativ** |
| ✅ | **Die Suppe** wird (von dem Koch) **gekocht**. | النتيجة النهائية |

**مثال آخر:**
- Aktiv: `Der Lehrer erklärt die Grammatik.` — المدرس يشرح القواعد.
- Passiv: `Die Grammatik **wird** (vom Lehrer) **erklärt**.` — القواعد تُشرح (من المدرس).

---

## ⚠️ Passiv مع Modalverben

لما نجيب Modalverb مع Passiv، التركيب يكون:

```
[Subject] + Modalverb (مصرف) + Partizip II + werden (في نهاية الجملة!)
```

| Modalverb | Aktiv | Passiv |
|-----------|-------|--------|
| **müssen** | Man muss die Tür schließen. | Die Tür **muss geschlossen werden**. |
| **können** | Man kann das Fenster öffnen. | Das Fenster **kann geöffnet werden**. |
| **sollen** | Man soll das Licht ausschalten. | Das Licht **soll ausgeschaltet werden**. |
| **dürfen** | Man darf hier nicht parken. | Hier **darf nicht geparkt werden**. |

📌 **انتبه:** `werden` يجي في **آخر الجملة** بشكل Infinitiv مش مصروف.

## ✅ جمل أمثلة مع ترجمة

| الجملة | الترجمة |
|--------|---------|
| Das Brot **wird gebacken**. | الخبز يُخبز. |
| Der Brief **wird geschrieben**. | الرسالة تُكتب. |
| Die Prüfung **wird korrigiert**. | الامتحان يُصحّح. |
| Das Auto **wird repariert**. | السيارة تُصلح. |
| Hier **wird Deutsch gesprochen**. | هنا يُتحدث بالألمانية. |
| Die Tür **muss geschlossen werden**. | الباب يجب أن يُغلق. |
| Das Fenster **kann geöffnet werden**. | النافذة يمكن فتحها. |
| Im Krankenhaus **darf nicht geraucht werden**. | في المستشفى لا يُسمح بالتدخين. |

## 📝 تمارين

### تمرين 1: حدد — Aktiv أم Passiv؟
1. Der Mann kauft ein Auto. → ___
2. Das Auto wird gekauft. → ___
3. Die Kinder spielen im Garten. → ___
4. Das Fenster wird geöffnet. → ___
5. Der Brief wird geschrieben. → ___

### تمرين 2: حول من Aktiv إلى Passiv
1. Der Bäcker backt das Brot. → Das Brot ___ ___.
2. Der Mechaniker repariert das Auto. → Das Auto ___ ___.
3. Die Mutter kocht das Essen. → Das Essen ___ ___.
4. Der Schüler liest das Buch. → Das Buch ___ ___.
5. Der Künstler malt das Bild. → Das Bild ___ ___.

### تمرين 3: أكمل بـ wird/werden/wird... (Passiv)
1. Das Wasser ___ getrunken.
2. Die Bücher ___ gelesen.
3. Der Kaffee ___ getrunken.
4. Die Blumen ___ gegossen.
5. Ich ___ gefragt.

### تمرين 4: Passiv مع Modalverb
1. Man muss die Aufgabe machen. → Die Aufgabe ___ ___ ___.
2. Man kann das Problem lösen. → Das Problem ___ ___ ___.
3. Man soll den Müll trennen. → Der Müll ___ ___ ___.
4. Man darf hier nicht rauchen. → Hier ___ nicht ___ ___.
5. Man muss die Tür schließen. → Die Tür ___ ___ ___.

### تمرين 5: صحح الخطأ (الجملة فيها خطأ)
1. Das Buch wird lesen. (غلط)
2. Die Suppe wird gekocht werden. (إضافي)
3. Der Brief wird von der Lehrer geschrieben. (غلط في Dativ)
4. Das Fenster kann geöffnet. (ناقص)
5. Die Kinder werden gefragt von der Mutter. (ترتيب خطأ)

## ✅ الحلول

### حل تمرين 1
1. **Aktiv** | 2. **Passiv** | 3. **Aktiv** | 4. **Passiv** | 5. **Passiv**

1. Das Brot **wird gebacken**.
2. Das Auto **wird repariert**.
3. Das Essen **wird gekocht**.
4. Das Buch **wird gelesen**.
5. Das Bild **wird gemalt**.

### حل تمرين 3
1. **wird** | 2. **werden** | 3. **wird** | 4. **werden** | 5. **werde**

### حل تمرين 4
1. Die Aufgabe **muss gemacht werden**.
2. Das Problem **kann gelöst werden**.
3. Der Müll **soll getrennt werden**.
4. Hier **darf nicht geraucht werden**.
5. Die Tür **muss geschlossen werden**.

### حل تمرين 5
1. Das Buch **wird gelesen**. (مش lesen)
2. Die Suppe **wird gekocht**. (شيل werden الزايد)
3. Der Brief wird **von der Lehrerin** / **vom Lehrer** geschrieben.
4. Das Fenster kann **geöffnet werden**. (أضف werden)
5. Die Kinder **werden von der Mutter gefragt**. (von der Mutter قبل gefragt أو قبله)

---

💡 **نصيحة سريعة:** الفرق بين Aktiv و Passiv هو التركيز. Aktiv = الفاعل مهم. Passiv = الفعل (أو النتيجة) مهم. في الامتحانات، أشهر سؤال هو تحويل Aktiv → Passiv. اتبع الخطوات الأربع فوق وستنجح!
