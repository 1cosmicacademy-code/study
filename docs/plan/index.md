# 🗓️ الخطة التعليمية — جدول التعلم

> هذه الخطة تأخذك من **الصفر** إلى **مستوى A1.2** خطوة بخطوة. كل أسبوع ندرس درس معين بالترتيب.

---

## 📅 كيف نستخدم الخطة؟

1. **كل أسبوع** — ندرس Lektion (درس) واحد
2. **بعد الدرس** — نحل التمارين في Übungsbuch
3. **كل أسبوعين** — نراجع القواعد المرتبطة بالدروس السابقة
4. **كل درس فيه** فيديو يوتيوب — عربي وإنجليزي
