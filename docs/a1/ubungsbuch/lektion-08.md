## 🧩 تمرين 1: املأ الفراغ — haben oder sein؟

|---|--------|------------|
| 1. | Ich _______ gestern Geburtstag _______________. (feiern) | |
| 2. | Meine Freunde _______ zur Party _______________. (kommen) | |
| 3. | Wir _______ Cola und Saft _______________. (trinken) | |

## 🧩 تمرين 3: ربط الجملة مع Partizip II

|--------|---|-------------|
| 1. Ich habe einen Kuchen ... | (أ) | getrunken |
| 2. Wir haben Musik ... | (ب) | gekommen |

| 5. Ihr seid nach Hause ... | (ج) | gehört |
| 6. Sie hat Pizza ... | (ح) | mitgebracht |

## 🧩 تمرين 6: أعد كتابة الجمل في الماضي (Perfekt)

| Präsens (المضارع) | Perfekt (الماضي) |
|-------------------|-----------------|

| 3. Wir singen ein Lied. | → ________________ |
| 4. Lisa tanzt viel. | → ________________ |

| # | السؤال | الإجابة |
|---|--------|---------|

| 5. | Habt ihr getanzt? (ja, viel) | |

## 🧩 تمرين 10: اكتب 3 جمل عن نفسك

1. ___________________________________

2. ___________________________________

3. ___________________________________
