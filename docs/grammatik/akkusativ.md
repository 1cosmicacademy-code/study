# حالة النصب (Akkusativ) في اللغة الألمانية

🎯 **حالة النصب** هي الحالة الثانية في الألمانية. نستخدمها بشكل أساسي للمفعول به المباشر — الشيء أو الشخص اللي يقع عليه الفعل.

---

## 💡 متى نستخدم Akkusativ؟

### 1. المفعول به المباشر (Direct Object)

اللي يجيب على سؤال **"wen?"** (مَن؟) أو **"was?"** (ماذا؟)

```
Ich sehe den Lehrer. → Wen sehe ich? → Den Lehrer.
أرى المدرس.
```

```
Er isst einen Apfel. → Was isst er? → Einen Apfel.
هو يأكل تفاحة.
```

### 2. بعد أفعال معينة

أفعال كثيرة تأخذ Akkusativ بدون حرف جر:

| الفعل (Akkusativ) | المعنى |
|-------------------|--------|
| **haben** | يملك |
| **essen** | يأكل |
| **trinken** | يشرب |
| **sehen** | يرى |
| **hören** | يسمع |
| **kaufen** | يشتري |
| **lieben** | يحب |
| **kennen** | يعرف |
| **lesen** | يقرأ |
| **schreiben** | يكتب |
| **brauchen** | يحتاج |
| **nehmen** | يأخذ |

### 3. بعد حروف الجر الخاصة بـ Akkusativ

| حرف الجر | المعنى |
|----------|--------|
| **durch** | عبر، خلال |
| **für** | من أجل |
| **gegen** | ضد |
| **ohne** | بدون |
| **um** | حول |
| **bis** | حتى |
| **entlang** | بمحاذاة |

**أمثلة:**
- `Ich gehe **durch den** Park.` — أمشي عبر الحديقة.
- `Das Geschenk ist **für dich**.` — الهدية من أجلك.
- `**Ohne dich** kann ich nicht leben.` — بدونك لا أستطيع العيش.

---

## ⚠️ تغيير أدوات التعريف في Akkusativ

| الحالة | der (مذكر) | die (مؤنث) | das (محايد) | die (جمع) |
|--------|-----------|------------|-------------|-----------|
| **Nominativ** | der | die | das | die |
| **Akkusativ** | **den** | die | das | die |

✅ **فقط المذكر (der) اللي يتغير:** der → den  
الباقي (die, das, die) يبقى زي ما هو.

### جدول الأدوات الكامل

| نوع الأداة | مذكر (m) | مؤنث (f) | محايد (n) | جمع (pl) |
|-----------|----------|----------|-----------|----------|
| **der →** | **den** | die | das | die |
| **ein →** | **einen** | eine | ein | --- |
| **kein →** | **keinen** | keine | kein | **keine** |
| **mein →** | **meinen** | meine | mein | **meine** |

---

## 🎯 قاعدة ذهبية

```
Akkusativ = المفعول به المباشر
المذكر فقط يتغير: der → den, ein → einen
```

## ✅ جمل أمثلة مع الترجمة

| الجملة | الترجمة |
|--------|---------|
| Ich habe **einen** Hund. | عندي كلب. |
| Sie isst **den** Kuchen. | هي تأكل الكعكة. |
| Wir kaufen **das** Auto. | نحن نشتري السيارة. |
| Er liebt **die** Musik. | هو يحب الموسيقى. |
| Ich brauche **einen** Kuli. | أحتاج قلماً. |
| Das Geschenk ist **für den** Vater. | الهدية للأب. |
| Wir gehen **durch den** Wald. | نمشي عبر الغابة. |
