> 🎯 **الهدف:** اختبر نفسك في أجزاء الجسم والألم والدكتور.  
> 💡 **نصيحة:** حاول تحل التمارين بنفسك أولاً، بعدين شوف الحل!

## التمرين 2: صِل (Match) 🔗

**صِل الألماني بالعربي:**

| الألمانية | العربية |
|-----------|---------|
| 1. der Hals | أ. الكتف |
| 2. die Schulter | ب. البطن |
| 3. der Arm | ج. الرقبة / الحلق |
| 4. der Bauch | د. الساق |
| 5. das Bein | ه. الذراع |
| 6. der Rücken | و. الركبة |
| 7. das Knie | ز. الظهر |
| 8. die Zähne | ح. الأسنان |

✅ **الحل:**

| # | الإجابة |
|---|---------|
| 1 | **ج** — der Hals = الرقبة / الحلق |
| 2 | **أ** — die Schulter = الكتف |
| 3 | **ه** — der Arm = الذراع |
| 4 | **ب** — der Bauch = البطن |
| 5 | **د** — das Bein = الساق |
| 6 | **ز** — der Rücken = الظهر |
| 7 | **و** — das Knie = الركبة |
| 8 | **ح** — die Zähne = الأسنان |

## التمرين 3: اختر mein أو meine 🎯

**اختر mein أو meine حسب أداة الكلمة:**

|---|--------|:----:|:-----:|

| 3 | **______** Bauch tut weh. (der) | ⬜ | ⬜ |
| 4 | **______** Nase tut weh. (die) | ⬜ | ⬜ |
| 5 | **______** Bein tut weh. (das) | ⬜ | ⬜ |
| 6 | **______** Füße tun weh. (جمع) | ⬜ | ⬜ |
| 7 | **______** Auge tut weh. (das) | ⬜ | ⬜ |

| # | الإجابة | الشرح |

| 7 | **mein** — Mein Auge | das Auge → mein |

## التمرين 4: أكمل الجملة — الأمراض 🏥

**أكمل الجملة بكلمة من الصندوق:**

> Fieber — Husten — Schnupfen — Kopfschmerzen — Halsschmerzen — Erkältung

| # | الجملة |
|---|--------|
| 1 | Ich habe _____________. (درجة حرارة عالية — 39°) |
| 2 | Ich habe _____________. (كحة / كفّة) |
| 3 | Ich habe _____________. (رشح / زكام — أنف يسيل) |
| 4 | Ich habe _____________. (ألم في الرأس — صداع) |
| 5 | Ich habe _____________. (ألم في الحلق) |
| 6 | Ich habe eine _____________. (نزلة برد — كل الأعراض مع بعض) |

✅ **الحل:**

| # | الإجابة | الترجمة |
|---|---------|---------|
| 1 | Ich habe **Fieber**. | عندي حرارة. |
| 2 | Ich habe **Husten**. | عندي كحة. |
| 3 | Ich habe **Schnupfen**. | عندي رشح / زكام. |
| 4 | Ich habe **Kopfschmerzen**. | عندي صداع. |
| 5 | Ich habe **Halsschmerzen**. | حلقي يؤلمني. |
| 6 | Ich habe eine **Erkältung**. | عندي نزلة برد. |

## التمرين 5: كوّن جملة (املأ الفراغ في المحادثة) 💬

**املأ الفراغ بالكلمة المناسبة:**

> **Patient:** Guten Tag, Herr Doktor!  
> **Arzt:** Guten Tag! Was fehlt ______________? (لك — رسمي)  
> **Patient:** Ich ______________ Kopfschmerzen. (عندي — تصريف haben)  
> **Arzt:** Haben Sie auch ______________? (حرارة)  
> **Patient:** Ja, 38 Grad.  
> **Arzt:** Machen Sie bitte den ______________ auf. (الفم)  
> **Patient:** Aaaaah.  
> **Arzt:** Gute ______________! (الشفاء العاجل)

✅ **الحل:**

| # | الإجابة | الشرح |
|---|---------|-------|
| 1 | **Ihnen** | Was fehlt Ihnen? = ما بك؟ (رسمي) |
| 2 | **habe** | Ich habe = أنا عندي |
| 3 | **Fieber** | حرارة / حمى |
| 4 | **Mund** | den Mund aufmachen = يفتح الفم |
| 5 | **Besserung** | Gute Besserung! = بالشفاء العاجل! |

| 5 | يدي تؤلمني. | ______________ |

## التمرين 7: صح أم خطأ؟ (Richtig oder falsch?) ✅❌

**هل الجملة صحيحة (✅) أم خاطئة (❌)؟**

| # | الجملة | ✅ / ❌ |
|---|--------|:-------:|
| 1 | Ich habe **Fieber** = عندي كحة. | |
| 2 | **Der Mund** = الفم. | |
| 3 | **Meine Kopf** tut weh. (الصحيح) | |
| 4 | **Gute Besserung** = بالشفاء العاجل. | |
| 5 | **Die Zähne** = الأسنان (مفرد). | |
| 6 | **Ich habe Schmerzen** = عندي آلام. | |
| 7 | **Das Bein** = الذراع. | |
| 8 | **Kopfschmerzen** = ألم في البطن. | |

✅ **الحل:**

| # | الإجابة | التصحيح |
|---|---------|---------|
| 1 | ❌ | Fieber = حرارة / حمى. Husten = كحة. |
| 2 | ✅ | der Mund = الفم. صح. |
| 3 | ❌ | Mein Kopf (der → mein, بدون e) |
| 4 | ✅ | Gute Besserung = بالشفاء العاجل. صح. |
| 5 | ❌ | Die Zähne = جمع (مفردها der Zahn). |
| 6 | ✅ | Schmerzen = آلام. صح. |
| 7 | ❌ | Das Bein = الساق / الرجل. Der Arm = الذراع. |
| 8 | ❌ | Kopfschmerzen = صداع (ألم الرأس). Bauchschmerzen = ألم البطن. |

## التمرين 8: تصريف haben 🧩

**أكمل الجدول — صرّف الفعل haben:**

| الضمير | haben |
|--------|:-----:|
| ich | ________ |
| du | ________ |
| er / sie / es | ________ |
| wir | ________ |
| ihr | ________ |
| sie / Sie | ________ |

✅ **الحل:**

| الضمير | haben |
|--------|:-----:|
| ich | **habe** |
| du | **hast** |
| er / sie / es | **hat** |
| wir | **haben** |
| ihr | **habt** |
| sie / Sie | **haben** |

**جمل للمساعدة:**
- Ich **habe** Fieber. — أنا عندي حرارة.
- Du **hast** Husten. — أنت عندك كحة.
- Er **hat** Kopfschmerzen. — هو عنده صداع.
- Wir **haben** Halsschmerzen. — نحن عندنا ألم في الحلق.
- Ihr **habt** Schnupfen. — أنتم عندكم زكام.
- Sie **haben** eine Erkältung. — هم / حضرتك عندك نزلة برد.

## التمرين 9: رتب الجملة 🧩

**رتب الكلمات لتكون جملة صحيحة:**

| # | الكلمات | الجملة الصحيحة |
|---|---------|----------------|
| 1 | tut / Kopf / weh / der / mir | **________________** |
| 2 | Fieber / habe / ich | **________________** |
| 3 | mein / weh / Bein / tut | **________________** |
| 4 | Termin / einen / brauche / ich | **________________** |
| 5 | tun / Zähne / meine / weh | **________________** |
| 6 | haben / Sie / auch / Fieber / ? | **________________** |

✅ **الحل:**

| # | الجملة الصحيحة | الترجمة |
|---|-----------------|---------|
| 1 | **Der Kopf tut mir weh.** | رأسي يؤلمني. |
| 2 | **Ich habe Fieber.** | عندي حرارة. |
| 3 | **Mein Bein tut weh.** | ساقي تؤلمني. |
| 4 | **Ich brauche einen Termin.** | أحتاج موعداً. |
| 5 | **Meine Zähne tun weh.** | أسناني تؤلمني. |
| 6 | **Haben Sie auch Fieber?** | هل عندك حرارة أيضاً؟ |

**تذكير:** في الألمانية، الفعل دائماً في المكان الثاني!  
مثلاً: Ich **habe** Fieber. / Der Kopf **tut** mir weh.
