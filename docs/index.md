## 🧪 اكتشف مستواك — اختبار فحص المستوى (Einstufungstest)

> 🎯 **هل أنت A1 أم A2 أم B1؟** اختبر مستواك الحقيقي الآن — 42 سؤالاً ذكية من بنك ~200 سؤال تحدد مستواك وتقدم لك توصيات مخصصة!

| الميزة | الوصف |
|--------|-------|
| 🎯 **تحديد دقيق للمستوى** | A1 · A2 · B1 — نتيجة فورية بعد 42 سؤالاً |
| 📊 **تحليل نقاط القوة والضعف** | مصنفة حسب المجال: مفردات، قواعد، تركيب جمل، تواصل، استماع، قراءة |
| 🎓 **توصيات مخصصة** | دروس وتمارين تناسب مستواك — مع روابط مباشرة للقواعد التي تحتاجها |
| 🔄 **أسئلة متجددة** | كل مرة تختبر — أسئلة مختلفة بتحديد عشوائي من بنك ~200 سؤال |
