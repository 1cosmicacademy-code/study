# The Nominative Case in German

🎯 The **Nominative** is the basic case in German. It is the natural case for the subject — the person or thing doing the action.

## 💡 When do we use the Nominative?

### 1. The Subject — answers "who?" or "what?"

The person or thing doing the action.

```
Der Mann schläft. → Wer schläft? → Der Mann.
The man is sleeping.
```

```
Das Kind spielt. → Was spielt? → Das Kind.
The child is playing.
```

### 2. After the verb "sein" (to be)

```
Ich bin ein Student. → I am a student.
Das ist ein Tisch. → This is a table.
```

### 3. After the verb "werden" (to become)

```
Er wird ein Lehrer. → He becomes a teacher.
```

### 4. In greetings and fixed expressions

```
Guten Morgen! → Good morning!
(Das ist) Schade! → That's a pity!
```

## ⚠️ Articles in the Nominative

| Article | Masculine (m) | Feminine (f) | Neuter (n) | Plural (pl) |
|---------|--------------|-------------|-----------|-------------|
| **Definite (the)** | **der** | **die** | **das** | **die** |
| **Indefinite (a/an)** | **ein** | **eine** | **ein** | --- |
| **Negative (no)** | **kein** | **keine** | **kein** | **keine** |

**Examples:**
- **Der** Mann ist alt. — The man is old.
- **Die** Frau ist jung. — The woman is young.
- **Das** Kind ist klein. — The child is small.
- **Ein** Hund bellt. — A dog is barking.
- **Keine** Zeit. — No time.

## 🎯 Golden Rule

```
Nominative = the subject = the one doing the action
Masculine article = der, Feminine = die, Neuter = das
```

## ✅ Example Sentences

| Sentence | Translation |
|----------|------------|
| **Der** Hund ist braun. | The dog is brown. |
| **Die** Katze ist schwarz. | The cat is black. |
| **Das** Haus ist groß. | The house is big. |

| **Ein** Mann kommt. | A man is coming. |
| **Eine** Frau singt. | A woman is singing. |
