# Funktionen von werden — Functions of the verb *werden*

## 🎯 Lesson Goals

- Recognise the four main uses of *werden*: **state**, **passive**, **process**, **development**.
- Apply the correct *werden* construction for each function.
- Avoid common pitfalls when mixing the uses.

---

## 📚 Core Rule

The verb **werden** can serve four distinct grammatical functions in German. It never changes the core meaning of the sentence but signals whether the clause expresses a *state*, a *passive voice*, a *future/action*, or a *developmental change*.

| Function | Meaning | Example (German) | English Translation |
|----------|---------|------------------|---------------------|
| **State** (Zustand) | Describes a new or changed state. | *Er ist müde **geworden**.* | He has become tired. |
| **Passive** (Passiv) | Forms the passive voice. | *Das Haus **wird** gebaut.* | The house is being built. |
| **Process / Future** (Vorgang) | Indicates a future action or process. | *Wir **werden** morgen reisen.* | We will travel tomorrow. |
| **Development** (Entwicklung) | Shows progressive development. | *Der Baum **wird** immer größer.* | The tree keeps getting bigger. |

### Typical Errors (Typische Fehler)
- Using **haben** instead of **sein** with *geworden* in a state sentence (e.g., *Er hat müde geworden*).
- Mixing passive and process forms: **wurde** vs. **wird** – *wurde* for past passive, **wird** for present/future.
- Forgetting **sein** when *geworden* is used; the auxiliary must match the subject.
- Using *werden* for a simple present when a developmental meaning is intended; add **immer** or **stets** to clarify.

---

## 🎯 Example Sentences

1. *Er ist müde **geworden**.* — "He has become tired."
2. *Das Auto **wird** repariert.* — "The car is being repaired."
3. *Wir **werden** morgen fahren.* — "We will travel tomorrow."
4. *Der Fluss **wird** breiter.* — "The river will become wider."
