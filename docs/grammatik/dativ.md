# حالة الجر (Dativ) في اللغة الألمانية

🎯 **حالة الجر (Dativ)** هي الحالة الثالثة في الألمانية. نستخدمها غالباً للمفعول به غير المباشر — الشخص المستفيد من الفعل.

---

## 💡 متى نستخدم Dativ؟

### 1. المفعول به غير المباشر (Indirect Object)

اللي يجيب على سؤال **"wem?"** (لمَن؟)

```
Ich gebe dem Kind einen Ball.
أعطي الطفل كرةً.
→ Wem gebe ich den Ball? → Dem Kind.
لمَن أعطي الكرة؟ للطفل. ←
```

🎯 **الطفل (Dem Kind)** هو Dativ (غير مباشر)  
**الكرة (Einen Ball)** هو Akkusativ (مباشر)

### 2. أفعال تأخذ Dativ دائماً (بدون Akkusativ)

| الفعل | المعنى | مثال |
|-------|--------|------|
| **helfen** | يساعد | Ich helfe **dem** Mann. |
| **danken** | يشكر | Wir danken **dir**. |
| **gefallen** | يعجب | Das gefällt **mir**. |
| **gehören** | يخص | Das gehört **dem** Lehrer. |
| **schmecken** | يطيب (طعم) | Die Pizza schmeckt **mir**. |
| **passen** | يناسب | Die Hose passt **mir**. |
| **folgen** | يتبع | Er folgt **dem** Hund. |
| **glauben** | يصدق | Ich glaube **dir**. |
| **antworten** | يجيب | Er antwortet **der** Frau. |
| **zuhören** | يستمع | Hör **mir** zu! |

### 3. حروف الجر اللي تأخذ Dativ دائماً

| حرف الجر | المعنى | مثال |
|----------|--------|------|
| **aus** | من (داخل) | **aus dem** Haus |
| **bei** | عند، بالقرب من | **bei der** Arbeit |
| **mit** | مع | **mit dem** Freund |
| **nach** | إلى (بلد)، بعد | **nach der** Schule |
| **seit** | منذ | **seit einem** Jahr |
| **von** | من (شخص)، عن | **von dem** Vater |
| **zu** | إلى (شخص/مكان) | **zu der** Oma |
| **gegenüber** | مقابل | **gegenüber dem** Bahnhof |

### 4. حروف جر متغيرة (Wechselpräpositionen) — Dativ للدلالة على المكان

حروف الجر: **in, an, auf, über, unter, neben, zwischen, vor, hinter**

نستخدم Dativ معها لما تحدد **المكان** (تسأل: Wo? = أين؟).

```
Ich bin in der Schule. → Wo? → Dativ
أنا في المدرسة.
```

```
Das Buch liegt auf dem Tisch. → Wo? → Dativ
الكتاب على الطاولة.
```

## ⚠️ تغيير أدوات التعريف في Dativ

| الحالة | der (مذكر) | die (مؤنث) | das (محايد) | die (جمع) |
|--------|-----------|------------|-------------|-----------|
| **Nominativ** | der | die | das | die |
| **Dativ** | **dem** | **der** | **dem** | **den** |

✅ **التغييرات:**
- der → **dem**
- die → **der**
- das → **dem**
- die (جمع) → **den** (+ تنضاف -n للاسم)

### جدول الأدوات الكامل في Dativ

| نوع الأداة | مذكر (m) | مؤنث (f) | محايد (n) | جمع (pl) |
|-----------|----------|----------|-----------|----------|
| **der/die/das →** | **dem** | **der** | **dem** | **den (+n)** |
| **ein/eine →** | **einem** | **einer** | **einem** | **---** |
| **kein →** | **keinem** | **keiner** | **keinem** | **keinen (+n)** |
| **mein →** | **meinem** | **meiner** | **meinem** | **meinen (+n)** |

⚠️ **قاعدة النون (n) للجمع:** في Dativ، الأسماء الجمع تضاف لها -n في النهاية إذا كانت مو منتهية بـ -n أو -s.

```
mit den Kind-er-n → مع الأطفال
von den Frau-en-n → من النساء
```

## 🎯 قواعد ذهبية

```
1. Dativ يأتي قبل Akkusativ في الجملة
   Ich gebe dem Mann das Buch.
2. أو يأتي بعد Akkusativ لو المفعولين ضمائر
   Ich gebe es ihm.
3. بعض الأفعال لا تأخذ إلا Dativ فقط
```

---

## ✅ جمل أمثلة مع الترجمة

| الجملة | الترجمة |
|--------|---------|
| Ich helfe **dem** Lehrer. | أساعد المدرس. |
| Wir danken **dir** für die Hilfe. | نشكرك على المساعدة. |
| Das Buch gehört **der** Frau. | الكتاب يخص المرأة. |
| Er wohnt **bei seinem** Bruder. | هو يسكن عند أخيه. |
| Ich fahre **mit dem** Bus. | أركب الباص. |
| **Seit einem** Jahr lerne ich Deutsch. | منذ سنة أتعلم الألمانية. |
| Das Geschenk ist **von meiner** Mutter. | الهدية من أمي. |

### تمرين 2: أكمل بـ aus, bei, mit, nach, seit, von, zu
1. Ich komme ___ der Schule. (من)
2. Er wohnt ___ seinen Eltern. (عند)
3. Wir fahren ___ dem Arzt. (إلى)
4. Sie arbeitet ___ einer Firma. (في/عند)
5. Ich lerne Deutsch ___ einem Jahr. (منذ)

### تمرين 3: اختر Dativ oder Akkusativ — ضع wo? (D) أو wohin? (A)
1. Ich bin in ___ Schule. (die)
2. Ich gehe in ___ Schule. (die)
3. Das Buch liegt auf ___ Tisch. (der)
4. Ich lege das Buch auf ___ Tisch. (der)
5. Sie sitzt neben ___ Freund. (der)

### تمرين 4: رتب الجملة — ضع Dativ قبل Akkusativ
1. Ich / geben / der Mann / das Buch
2. Er / kaufen / seine Frau / ein Geschenk
3. Sie / zeigen / der Tourist / die Stadt
4. Wir / schicken / der Freund / eine Karte
5. Der Vater / bringen / das Kind / ein Spielzeug

### تمرين 5: صحح الخطأ
1. Ich helfe der Mann.
2. Das Buch gehört den Kind.
3. Wir danken dem Lehrerin.
4. Er fährt mit der Auto.
5. Sie wohnt seit einen Jahr hier.

## ✅ الحلول

### حل تمرين 1
1. dem | 2. der | 3. dem | 4. der | 5. dem

### حل تمرين 2
1. aus | 2. bei | 3. zu | 4. bei (oder in einer) | 5. seit

### حل تمرين 3
1. D → der | 2. A → die | 3. D → dem | 4. A → den | 5. D → dem

### حل تمرين 4
1. Ich gebe dem Mann das Buch.
2. Er kauft seiner Frau ein Geschenk.
3. Sie zeigt dem Touristen die Stadt.
4. Wir schicken dem Freund eine Karte.
5. Der Vater bringt dem Kind ein Spielzeug.

### حل تمرين 5
1. dem Mann (der → dem) | 2. dem Kind (den → dem) | 3. der Lehrerin (dem → der) | 4. dem Auto (der → dem) | 5. einem Jahr (einen → einem)
