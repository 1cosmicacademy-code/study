## 2. 🟠 Choice/Text Type مع إجابات كلمات (Pattern B)

### 2.1 A1 Übungsbuch Lektion 19 — Übung 2
| الحقل | القيمة |
|-------|--------|
| **المسار** | `/a1/ubungsbuch/lektion-19/` |
| **العنوان** | 🥈 التمرين 2: اختر الإجابة الصحيحة |
| **نوع data-type** | `match` ❌ (يجب أن يكون `choice` أو `text`) |
| **data-answers** | `['die U-Bahn', 'das Fahrrad', 'das Flugzeug', 'der Zug', 'das Auto']` |
| **أعمدة الجدول** | # \| الجملة \| أ \| ب \| ج |
| **data-options** | غير موجود |

**الوصف:** التمرين من نوع `match` لكن إجاباته كلمات ألمانية كاملة. الجدول يحتوي أعمدة أ/ب/ج (عربي) مما يدل أنه من المفترض أن يكون `choice` type مع إجابات A/B/C. بدون `data-options`، القائمة المنسدلة لا تعمل.

**الإصلاح المقترح:** تحويل إلى `choice` type مع إجابات A/B/C، أو إزالة `data-type` ليصبح text مع بقاء الإجابات ككلمات ألمانية.

## 4. 🟠 A2 Kursbuch Lektion 01 — Übung 2

| الحقل | القيمة |
|-------|--------|
| **المسار** | `/a2/kursbuch/lektion-01/` |
| **المسار الإنجليزي** | `/a2/kursbuch/lektion-01.en/` |
| **Übung 2 type** | `text` |
| **data-answers** | `['links', 'rechts', 'geradeaus', 'Ampel']` |

### المشكلة 1 — السؤال 3: ↑ الإجابة مكشوفة
- السؤال: `↑ (geradeaus) = ___`
- التلميح (↑) مع كلمة "geradeaus" يعطي الإجابة مباشرة للطالب
- لا يحتاج الطالب إلى تفكير — فقط ينسخ الكلمة

### المشكلة 2 — السؤال 4: خلط لغات
- السؤال: `Die ___ (traffic light) ist rot.`
- التلميح بالإنجليزية "traffic light" وليس الألمانية
- التمرين بالكامل باللغة الألمانية — لماذا التلميح ؟بالإنجليزية
- الإجابة الصحيحة: `Ampel`
- التأثير: طالب عربي (الجمهور المستهدف) لا يستفيد من التلميح الإنجليزي، بل يرتبك

**الإصلاح المقترح:**
- Q3: غير التلميح ليصبح أقل وضوحاً — مثلاً `↔ (___ ) = geradeaus`
- Q4: استبدل "traffic light" بـ "إشارة المرور" (عربي) أو أزل التلميح كلياً
