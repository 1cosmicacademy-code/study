# Fragen bilden — تكوين الأسئلة

[🔙 العودة إلى القواعد](../index.md)

## 🎯 أهداف الدرس

- تعلم كيفية تشكيل أسئلة **W-Fragen** (مِـ) وَ **Ja/Nein-Fragen** (نعم/لا).
- تفهم ميزان الإستفهام والنفذ.
- تطبيق القاعد في جمل يومية.

---

## 📚 القاعدة الأساسية

### 1️⃣ **W‑Fragen** (أسئلة بأدوات الاستفهام)

| سؤال (W‑Frage) | معنى بالعربية |
|----------------|----------------|
| **Wer?** | مَن؟ |
| **Was?** | ما؟ |
| **Wo?**  | أين؟ |
| **Wann?**| متى؟ |
| **Wie?**| كيف؟ |
| **Warum?**| لماذا؟ |
| **Wohin?**| إلى أين؟ |
| **Woher?**| من أين؟ |

**التركيب**: **W‑Frage + فعل + فاعل + ...**

_مثال_: **Wer** bist du? — مَن أنت؟

### 2️⃣ **Ja/Nein‑Fragen** (أسئلة نعم/لا)

- تبدأ **بالفعل** مباشرةً متبوعاً بالفاعل.
- إذا كان الفعل مساعداً (**sein**, **haben**, **modalverben**) يأتي قبل الفاعل.

_مثال_: **Bist du müde?** — أَنتَ متعب؟

_مثال_: **Hast du Zeit?** — هل لديك وقت؟

## ✅ أمثلة وترجمة

| الجملة الألمانية | الترجمة العربية |
|-------------------|-------------------|
| **Wer** kommt heute? | مَن يأتي اليوم؟ |
| **Was** machst du? | ما الذي تفعله؟ |
| **Wo** ist das Buch? | أين الكتاب؟ |
| **Bist** du glücklich? | أَنتَ سعيد؟ |
| **Hast** du Hunger? | هل أنت جائع؟ |

## ✅ خلاصة

- **W‑Fragen** تبدأ بأداة سؤال ثم الفعل ثم الفاعل.
- **Ja/Nein‑Fragen** يبدأ الفعل (أحياناً مساعد) ثم الفاعل.
- تدرب على تحويل الجمل اليومية إلى أسئلة لتقوية الفهم.
