# Plural Forms in German

🎯 Forming plurals in German doesn't follow just one rule — but there are recurring patterns. The key: **learn the plural with each noun!**

---

## 💡 Main Plural Patterns

### 1. Add -e (most common)

| Singular | Plural | Meaning |
|----------|-------|---------|
| der Tisch | die Tisch**e** | table |
| der Hund | die Hund**e** | dog |
| der Tag | die Tag**e** | day |

### 2. Add -er (many neuter nouns)

| Singular | Plural | Meaning |
|----------|-------|---------|
| das Kind | die Kind**er** | child |
| das Buch | die Büch**er** | book (with Umlaut) |
| das Haus | die Häus**er** | house |

### 3. Add -n/-en (mostly feminine)

| Singular | Plural | Meaning |
|----------|-------|---------|
| die Frau | die Frau**en** | woman |
| die Blume | die Blume**n** | flower |
| die Tasche | die Tasche**n** | bag |

### 4. No ending + Umlaut (often masculine)

| Singular | Plural | Meaning |
|----------|-------|---------|
| der Vater | die V**ä**ter | father |
| der Apfel | die Äpfel | apple |
| die Mutter | die M**ü**tter | mother |

### 5. Add -s (foreign words)

| Singular | Plural | Meaning |
|----------|-------|---------|
| das Auto | die Auto**s** | car |
| der Park | die Park**s** | park |
| das Handy | die Handy**s** | mobile phone |

## ⚠️ Article with Plurals

All plurals take **die** in the Nominative:
- der Hund → **die** Hunde
- das Kind → **die** Kinder
- die Frau → **die** Frauen

## 📌 Golden Rule

```
Learn each noun with its plural from the start!
der Tisch → die Tische (not just Tisch)
```

## 📝 Exercises

### Exercise 1: Form the plural

1. der Hund → die ___
2. die Blume → die ___
3. das Kind → die ___
4. der Tisch → die ___
5. das Auto → die ___

### Exercise 2: Which plural is correct?

1. der Mann → die (Manner / Männer / Männern)
2. das Haus → die (Häuser / Hauser / Häusern)
3. die Mutter → die (Mütter / Mutter / Mutters)
4. der Student → die (Studenten / Student / Studentin)
5. das Buch → die (Bücher / Bucher / Büchen)

## ✅ Solutions

### Solution Exercise 1
1. Hunde | 2. Blumen | 3. Kinder | 4. Tische | 5. Autos

### Solution Exercise 2
1. Männer | 2. Häuser | 3. Mütter | 4. Studenten | 5. Bücher
