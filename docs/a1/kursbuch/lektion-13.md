## 1. 🧍 أجزاء الجسم (Körperteile)

الجسم فيه أجزاء كثيرة. كل جزء له أداة (der / die / das). مهم جداً تحفظ الأداة مع الكلمة!

| الألمانية | النطق (بالعربي) | العربية |
|-----------|-----------------|---------|
| **der Kopf** | **دير كوبف** — deer kopf | الرأس |
| **das Gesicht** | **داس غزيزت** — dahs guh-ZEEKT | الوجه |
| **das Auge** | **داس آوغه** — dahs OW-guh | العين |
| **die Nase** | **دي نازه** — dee NAH-zuh | الأنف |
| **der Mund** | **دير مونت** — deer moont | الفم |
| **das Ohr** | **داس أور** — dahs ohr | الأذن |
| **der Hals** | **دير هالس** — deer hahls | الرقبة / الحلق |
| **die Schulter** | **دي شولتر** — dee SHOOL-ter | الكتف |
| **der Arm** | **دير آرم** — deer arm | الذراع |
| **die Hand** | **دي هانت** — dee hahnt | اليد |
| **der Finger** | **دير فينغر** — deer FING-er | الإصبع |
| **der Bauch** | **دير باوخ** — deer bowkh | البطن |
| **der Rücken** | **دير روكِن** — deer RÜK-en | الظهر |
| **das Bein** | **داس باين** — dahs bine | الساق / الرجل |
| **das Knie** | **داس كني** — dahs knee | الركبة |
| **der Fuß** | **دير فوس** — deer fooss | القدم |
| **die Zähne** | **دي تسينه** — dee TSAY-nuh | الأسنان (جمع) |

> 💡 **نصيحة:** الأجزاء المفردة من الجسم مؤنثة (die) إذا كانت مزدوجة طبيعياً — die Hand, die Schulter. لكن فيه استثناءات!

---

## 2. 🏥 عند الطبيب — جمل مفيدة (Beim Arzt — Nützliche Sätze)

لما تروح للدكتور بالألماني، تحتاج تعبر عن الألم. أشهر جملة: **"X tut mir weh"** يعني "X يؤلمني".

### كيف تقول "يؤلمني"؟

| الألمانية | النطق | المعنى |
|-----------|-------|--------|
| **Der Kopf tut mir weh.** | **دير كوبف توت مير فيه** — deer kopf toot meer veh | رأسي يؤلمني. |
| **Der Bauch tut mir weh.** | **دير باوخ توت مير فيه** — deer bowkh toot meer veh | بطني يؤلمني. |
| **Die Hand tut mir weh.** | **دي هانت توت مير فيه** — dee hahnt toot meer veh | يدي تؤلمني. |
| **Das Bein tut mir weh.** | **داس باين توت مير فيه** — dahs bine toot meer veh | ساقي يؤلمني. |
| **Die Zähne tun mir weh.** | **دي تسينه تون مير فيه** — dee TSAY-nuh toon meer veh | أسناني تؤلمني. |

### الأمراض والحالات

| الألمانية | النطق | المعنى |
|-----------|-------|--------|
| **Ich habe Fieber.** | **إيخ هابه فيبر** — ich HAH-buh FEE-ber | عندي حرارة / حمى. |
| **Ich habe Husten.** | **إيخ هابه هوستن** — ich HAH-buh HOO-sten | عندي كحة. |
| **Ich habe Schnupfen.** | **إيخ هابه شنوبفن** — ich HAH-buh SHNOOP-fen | عندي زكام / رشح. |
| **Ich habe Kopfschmerzen.** | **إيخ هابه كوبفشمرتسن** — ich HAH-buh KOPF-shmair-tsen | عندي صداع. |
| **Ich habe Halsschmerzen.** | **إيخ هابه هالسشمرتسن** — ich HAH-buh HAHLS-shmair-tsen | حلقي يؤلمني. |
| **Mir ist kalt.** | **مير إيست كالت** — meer ist kalt | أنا أشعر بالبرد. |
| **Mir ist heiß.** | **مير إيست هايس** — meer ist hyce | أنا أشعر بالحر. |

> ⚠️ **قاعدة:** مع **wehtun** نستخدم **Dativ** — **mir** (لي / أنا).  
> Der Kopf tut **mir** weh. = رأسي يؤلمني (حرفياً: الرأس يؤلم **لي**).

## 3. 📝 القاعدة: haben + Schmerzen + der/mein (Grammar)

### 3.1 الفعل "haben" (يملك / عنده)

الفعل **haben** يتغير مع الضمائر. احفظه كويس لأنه أكثر فعل استخداماً بعد "sein".

| الضمير | التصريف | النطق | مثال |
|--------|---------|-------|------|
| **ich** | **habe** | **هابه** — HAH-buh | Ich **habe** Fieber. |
| **du** | **hast** | **هاست** — hahst | Du **hast** Husten. |
| **Sie** (رسمي) | **haben** | **هابن** — HAH-ben | Sie **haben** Schnupfen. |
| **er / sie** | **hat** | **هات** — haht | Er **hat** Kopfschmerzen. |
| **wir** | **haben** | **هابن** — HAH-ben | Wir **haben** Fieber. |
| **ihr** | **habt** | **هابت** — hahbt | Ihr **habt** Halsschmerzen. |
| **sie** (هم) | **haben** | **هابن** — HAH-ben | Sie **haben** Schmerzen. |

### 3.2 mein / meine (ضمائر الملكية مع أجزاء الجسم)

في الألمانية، نقول **"mein Kopf"** (رأسي) أو **"meine Hand"** (يدي). الأداة تحدد:

| الأداة الأصلية | mein + أداة |
|---------------|-------------|
| **der** (مذكر) | **mein** — mein Kopf, mein Arm, mein Bauch |
| **die** (مؤنث) | **meine** — meine Hand, meine Nase, meine Schulter |
| **das** (محايد) | **mein** — mein Bein, mein Auge, mein Ohr |
| **die** (جمع) | **meine** — meine Zähne, meine Augen, meine Ohren |

**أمثلة:**
- **Mein Kopf tut weh.** — رأسي يؤلمني.
- **Meine Hand tut weh.** — يدي تؤلمني.
- **Mein Bein tut weh.** — ساقي يؤلمني.
- **Meine Zähne tun weh.** — أسناني تؤلمني.

> 🎯 **لاحظ:** مع die المفرد ومع الجمع نستخدم **meine**، ومع der/das نستخدم **mein**.

## 4. 💬 محادثة (Gespräch — Beim Arzt)

> **Patient:** Guten Tag, Herr Doktor!  
> *(صباح الخير يا دكتور!)*
> 
> **Arzt:** Guten Tag! Was fehlt Ihnen?  
> *(صباح الخير! ما بك / ما مشكلتك؟)*
> 
> **Patient:** Ich habe Kopfschmerzen und der Hals tut mir weh.  
> *(عندي صداع وحلقي يؤلمني.)*
> 
> **Arzt:** Haben Sie auch Fieber?  
> *(هل عندك حرارة أيضاً؟)*
> 
> **Patient:** Ja, ich habe Fieber. 39 Grad.  
> *(نعم، عندي حرارة. 39 درجة.)*
> 
> **Arzt:** Gut, ich untersuche Sie. Machen Sie bitte den Mund auf.  
> *(حسناً، سأفحصك. افتح فمك من فضلك.)*
> 
> **Patient:** Aaaaah...  
> *(آآآه...)*
> 
> **Arzt:** Ja, der Hals ist rot. Sie haben eine Erkältung. Ich gebe Ihnen ein Rezept.  
> *(نعم، الحلق أحمر. عندك نزلة برد. سأعطيك روشتة.)*
> 
> **Patient:** Vielen Dank, Herr Doktor!  
> *(شكراً جزيلاً يا دكتور!)*
> 
> **Arzt:** Bitte schön! Gute Besserung!  
> *(عفواً! بالشفاء العاجل!)*

### كلمات من الحوار:

| الكلمة | النطق | المعنى |
|--------|-------|--------|
| **Was fehlt Ihnen?** | **فاس فيلت إينن** — vahs feelt EE-nen | ما بك؟ |
| **untersuchen** | **أونتيرزوخن** — oon-ter-ZOO-khen | يفحص |
| **den Mund aufmachen** | **دين مونت آوفماخن** — deen moont OWF-mah-khen | يفتح الفم |
| **rot** | **روت** — roht | أحمر |
| **die Erkältung** | **دي إركيلتونغ** — dee er-KEL-toong | نزلة برد |
| **das Rezept** | **داس ريتسفت** — dahs reh-TSEPT | روشتة / وصفة طبية |
| **Gute Besserung!** | **غوته بيسرونغ** — GOO-teh BES-er-oong | بالشفاء العاجل! |

## 6. 🧠 تذكّر سريع (Schnelle Erinnerung)

- أجزاء الجسم لها أدوات: **der** (مذكر)، **die** (مؤنث)، **das** (محايد)
- الألم: **X tut mir weh** (X يؤلمني)
- الأمراض: **Ich habe + اسم المرض** (عندي ...)
- الدعاء للمريض: **Gute Besserung!** (بالشفاء!)
- **mein** مع der/das، **meine** مع die/الجمع
