# حالة النصب (Akkusativ) في اللغة الألمانية

🎯 **حالة النصب** هي الحالة الثانية في الألمانية. نستخدمها بشكل أساسي للمفعول به المباشر — الشيء أو الشخص اللي يقع عليه الفعل.

---

## 💡 متى نستخدم Akkusativ؟

### 1. المفعول به المباشر (Direct Object)

اللي يجيب على سؤال **"wen?"** (مَن؟) أو **"was?"** (ماذا؟)

```
Ich sehe den Lehrer. → Wen sehe ich? → Den Lehrer.
أرى المدرس.
```

```
Er isst einen Apfel. → Was isst er? → Einen Apfel.
هو يأكل تفاحة.
```

### 2. بعد أفعال معينة

أفعال كثيرة تأخذ Akkusativ بدون حرف جر:

| الفعل (Akkusativ) | المعنى |
|-------------------|--------|
| **haben** | يملك |
| **essen** | يأكل |
| **trinken** | يشرب |
| **sehen** | يرى |
| **hören** | يسمع |
| **kaufen** | يشتري |
| **lieben** | يحب |
| **kennen** | يعرف |
| **lesen** | يقرأ |
| **schreiben** | يكتب |
| **brauchen** | يحتاج |
| **nehmen** | يأخذ |

### 3. بعد حروف الجر الخاصة بـ Akkusativ

| حرف الجر | المعنى |
|----------|--------|
| **durch** | عبر، خلال |
| **für** | من أجل |
| **gegen** | ضد |
| **ohne** | بدون |
| **um** | حول |
| **bis** | حتى |
| **entlang** | بمحاذاة |

**أمثلة:**
- `Ich gehe **durch den** Park.` — أمشي عبر الحديقة.
- `Das Geschenk ist **für dich**.` — الهدية من أجلك.
- `**Ohne dich** kann ich nicht leben.` — بدونك لا أستطيع العيش.

---

## ⚠️ تغيير أدوات التعريف في Akkusativ

| الحالة | der (مذكر) | die (مؤنث) | das (محايد) | die (جمع) |
|--------|-----------|------------|-------------|-----------|
| **Nominativ** | der | die | das | die |
| **Akkusativ** | **den** | die | das | die |

✅ **فقط المذكر (der) اللي يتغير:** der → den  
الباقي (die, das, die) يبقى زي ما هو.

### جدول الأدوات الكامل

| نوع الأداة | مذكر (m) | مؤنث (f) | محايد (n) | جمع (pl) |
|-----------|----------|----------|-----------|----------|
| **der →** | **den** | die | das | die |
| **ein →** | **einen** | eine | ein | --- |
| **kein →** | **keinen** | keine | kein | **keine** |
| **mein →** | **meinen** | meine | mein | **meine** |

---

## 🎯 قاعدة ذهبية

```
Akkusativ = المفعول به المباشر
المذكر فقط يتغير: der → den, ein → einen
```

## ✅ جمل أمثلة مع الترجمة

| الجملة | الترجمة |
|--------|---------|
| Ich habe **einen** Hund. | عندي كلب. |
| Sie isst **den** Kuchen. | هي تأكل الكعكة. |
| Wir kaufen **das** Auto. | نحن نشتري السيارة. |
| Er liebt **die** Musik. | هو يحب الموسيقى. |
| Ich brauche **einen** Kuli. | أحتاج قلماً. |
| Das Geschenk ist **für den** Vater. | الهدية للأب. |
| Wir gehen **durch den** Wald. | نمشي عبر الغابة. |

## 📝 تمارين

### تمرين 1: أكمل بالأداة الصحيحة (der → den / ein → einen / die / das / keine)

1. Ich habe ___ Hund. (ein)
2. Sie kauft ___ Blumen. (die)
3. Wir sehen ___ Film. (der)
4. Er trinkt ___ Wasser. (das)
5. Ich brauche ___ Hilfe. (die)

### تمرين 2: أكمل بـ für, durch, ohne, um, gegen

1. Das ist ___ dich. (من أجلك)
2. Wir gehen ___ den Park. (عبر)
3. ___ dich gehe ich nicht. (بدونك)
4. Wir laufen ___ den See. (حول)
5. Das Medikament ist ___ Kopfschmerzen. (ضد)

### تمرين 3: حلل الجمل — حدد Akkusativ-Objekt

1. Ich lese ein Buch.
2. Der Vater kauft einen Ball.
3. Wir hören die Musik.
4. Das Kind trinkt Milch.
5. Ich kenne diesen Mann nicht.

### تمرين 4: حول من Nominativ إلى Akkusativ — غير الأداة

1. Der Tisch ist groß. → Ich sehe ___ Tisch.
2. Ein Hund bellt. → Ich höre ___ Hund.
3. Die Blume ist schön. → Ich kaufe ___ Blume.
4. Das Kind spielt. → Ich sehe ___ Kind.
5. Kein Student ist da. → Ich kenne ___ Studenten.

### تمرين 5: أكمل الجملة باختيار الصحيح

1. Ich habe (ein / einen / eine) Bruder.
2. Sie liebt (der / den / die) Mann.
3. Wir besuchen (das / den / die) Oma.
4. Er kauft (ein / eine / einen) Auto.
5. Ich möchte (eine / einen / ein) Kaffee, bitte.
