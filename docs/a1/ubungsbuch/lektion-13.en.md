## Exercise 5: Fill in the blanks — Doctor Dialogue 💬

**Fill in the blanks with the correct word:**

> **Patient:** Guten Tag, Herr Doktor!  
> **Arzt:** Guten Tag! Was fehlt ______________? (you — formal)  
> **Patient:** Ich ______________ Kopfschmerzen. (I have — conjugate haben)  
> **Arzt:** Haben Sie auch ______________? (fever)  
> **Patient:** Ja, 38 Grad.  
> **Arzt:** Machen Sie bitte den ______________ auf. (the mouth)  
> **Patient:** Aaaaah.  
> **Arzt:** Gute ______________! (get well soon)

✅ **Solution:**

| # | Answer | Explanation |
|---|--------|-------------|
| 1 | **Ihnen** | Was fehlt Ihnen? = What's wrong with you? (formal) |
| 2 | **habe** | Ich habe = I have |
| 3 | **Fieber** | fever |
| 4 | **Mund** | den Mund aufmachen = to open the mouth |
| 5 | **Besserung** | Gute Besserung! = Get well soon! |

## Exercise 7: True or False? (Richtig oder falsch?) ✅❌

**Is the sentence correct (✅) or wrong (❌)?**

| # | Sentence | ✅ / ❌ |
|---|----------|:-------:|
| 1 | Ich habe **Fieber** = I have a cough. | |
| 2 | **Der Mund** = the mouth. | |
| 3 | **Meine Kopf** tut weh. (correct form) | |
| 4 | **Gute Besserung** = Get well soon. | |
| 5 | **Die Zähne** = the tooth (singular). | |
| 6 | **Ich habe Schmerzen** = I am in pain. | |
| 7 | **Das Bein** = the arm. | |
| 8 | **Kopfschmerzen** = stomach ache. | |

✅ **Solution:**

| # | Answer | Correction |
|---|--------|------------|

| 6 | ✅ | Schmerzen = pain. Correct. |

## Exercise 8: Conjugate haben 🧩

**Complete the table — conjugate the verb haben:**

| Pronoun | haben |
|---------|:-----:|
| ich | ________ |
| du | ________ |
| er / sie / es | ________ |
| wir | ________ |
| ihr | ________ |
| sie / Sie | ________ |

✅ **Solution:**

| Pronoun | haben |
|---------|:-----:|
| ich | **habe** |
| du | **hast** |
| er / sie / es | **hat** |
| wir | **haben** |
| ihr | **habt** |
| sie / Sie | **haben** |

**Example sentences:**
- Ich **habe** Fieber. — I have a fever.
- Du **hast** Husten. — You have a cough.
- Er **hat** Kopfschmerzen. — He has a headache.
- Wir **haben** Halsschmerzen. — We have a sore throat.
- Ihr **habt** Schnupfen. — You (plural) have a runny nose.
- Sie **haben** eine Erkältung. — They / You (formal) have a cold.

## Exercise 9: Sentence Order 🧩

**Put the words in the correct order to make a sentence:**

| # | Words | Correct Sentence |
|---|-------|------------------|
| 1 | tut / Kopf / weh / der / mir | **________________** |
| 2 | Fieber / habe / ich | **________________** |
| 3 | mein / weh / Bein / tut | **________________** |
| 4 | Termin / einen / brauche / ich | **________________** |
| 5 | tun / Zähne / meine / weh | **________________** |
| 6 | haben / Sie / auch / Fieber / ? | **________________** |

✅ **Solution:**

| # | Correct Sentence | Translation |
|---|------------------|-------------|
| 1 | **Der Kopf tut mir weh.** | My head hurts. |
| 2 | **Ich habe Fieber.** | I have a fever. |
| 3 | **Mein Bein tut weh.** | My leg hurts. |
| 4 | **Ich brauche einen Termin.** | I need an appointment. |
| 5 | **Meine Zähne tun weh.** | My teeth hurt. |
| 6 | **Haben Sie auch Fieber?** | Do you also have a fever? |

**Remember:** In German, the verb is always in the second position!  
Example: Ich **habe** Fieber. / Der Kopf **tut** mir weh.
