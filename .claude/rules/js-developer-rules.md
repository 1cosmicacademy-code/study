# JS Developer — قواعد العمل

## القواعد الأساسية

1. **أقل تعديل ممكن** — لا تعيد كتابة الدوال الكاملة. أصلح السطر أو السطرين المسببين للمشكلة فقط.
2. **افهم السياق أولاً** — اقرأ الملف كاملاً قبل التعديل. افهم تدفق البيانات وكيف تعمل الدالة.
3. **اختبر منطقياً** — قبل الإعلان عن الإصلاح، تتبع منطق الكود ذهنياً وتأكد من صحة النتيجة.
4. **وثِّق التغيير** — أضف تعليقاً بالعربية أو الإنجليزية يشرح ما تم تغييره ولماذا.
5. **لا تكسر شيئاً آخر** — تأكد من أن إصلاحك لا يؤثر على أجزاء أخرى من النظام.

## Must / Must Not

- **يجب (Must):**
  - قراءة الملف كاملاً قبل التعديل
  - فهم بنية البيانات التي تتعامل معها الدالة
  - اختبار الإصلاح منطقياً (trace through the logic)
  - إضافة تعليق يشرح التغيير
  - التأكد من أن الإصلاح يعمل مع all edge cases (n=1, shift=0, إلخ)
  - التحقق من أن Console لن يظهر أي error جديد

- **ممنوع (Must Not):**
  - تعديل أكثر من اللازم (لا تعيد هيكلة الكود)
  - ترك dead code أو commented-out code
  - تعديل ملفات المحتوى (`*.md`)
  - افتراض أن الإجابة الأصلية دائماً 'A' (هذا هو الـ bug نفسه)
  - تجاهل الـ Console errors بعد الإصلاح

## معايير الجودة

### 1. دقة الإصلاح (Fix Accuracy)
- الإصلاح يعالج السبب الجذري وليس العرض فقط
- لا يغير سلوك الأجزاء السليمة من الكود
- **مؤشر الأداء:** 100% من حالات الخطأ تُعالج

### 2. جودة الكود (Code Quality)
- لا أكواد مكررة أو مهملة
- تعليقات واضحة ومفيدة
- استخدام نفس نمط الكود الموجود
- **مؤشر الأداء:** كود نظيف بمستوى الكود الأصلي أو أفضل

### 3. الانحدار (Regression)
- 0% أخطاء جديدة ناتجة عن الإصلاح
- **مؤشر الأداء:** لا عودة لأخطاء سابقة

## المعرفة المطلوبة

### فهم `shuffleOptions()` (level-test.js)
```
المشكلة: الدالة تفترض أن qd.answer دائماً 'A' (index 0)
         ثم تحسب الإجابة الجديدة: newIdx = (n - shift) % n
         هذا صحيح فقط لو الإجابة الأصلية index 0

الحل:    اقرأ qd.answer الأصلي ('A', 'B', 'C')
         حوّله إلى index: origIdx = OPTION_LABELS.indexOf(qd.answer)
         طبّق shift: newIdx = (origIdx + shift) % n
         ثم qd.answer = OPTION_LABELS[newIdx]
```

### فهم `video-switcher.js`
```
الدالة تبحث عن h2 فيه "فيديو"/"Video" ثم تُنشئ iframe
مشاكل محتملة: h2 غير موجود، videos.json لا يُحمّل،
              استثناء JS يوقف التنفيذ، ترتيب التحميل
```

## قائمة التحقق قبل التسليم

- [ ] هل الإصلاح يعالج السبب الجذري؟
- [ ] هل تم اختبار الإصلاح منطقياً؟
- [ ] هل جميع الحالات الحدية مغطاة؟
- [ ] هل هناك تعليق يشرح التغيير؟
- [ ] هل Console خالٍ من الأخطاء الجديدة؟
- [ ] هل تم تعديل أقل عدد ممكن من الأسطر؟
