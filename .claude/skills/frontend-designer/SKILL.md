# مهارة /frontend-designer

استخدم هذه المهارة عندما تحتاج إلى:
- تصميم واجهات مستخدم جديدة
- مراجعة وتدقيق تصميمات موجودة
- تحسين تجربة المستخدم وتفاعلاته
- إضافة حركات وتحسين أدائها
- تحسين الألوان والطباعة والتخطيط
- تطبيق معايير الوصول والأداء

## خطوات التنفيذ

### الخطوة 1: تشغيل الوكيل
استخدم الأمر التالي لتشغيل وكيل مصمم الواجهات:

```
Agent مخصص لمهام التصميم
```

### الخطوة 2: تحديد نطاق المهمة
عند تشغيل الوكيل، حدد له بوضوح:
1. **الوضع** — أي وضع تشغيل من أوضاع التصميم (design, audit, polish, animate, color, review, layout, type, accessibility, ux)
2. **النطاق** — الملفات أو المكونات المستهدفة
3. **المعايير** — معايير القبول للنتيجة النهائية

### الخطوة 3: المراجعة
بعد الانتهاء، راجع المخرجات وتأكد من:
- اكتمال المهمة
- جودة النتائج وفقاً لمعايير التصميم
- عدم وجود مشاكل في الوصول أو الأداء

## مثال استخدام

```
/frontend-designer صمم واجهة تسجيل دخول متطورة
    - الوضع: design
    - المطلوب: صفحة كاملة HTML/CSS
    - المعايير: متجاوب، متوافق مع WCAG AA، أداء عالي
```
