# Subordinate Clauses (Nebensätze) in German

🎯 **Subordinate clauses (Nebensätze)** are clauses that depend on a main clause (Hauptsatz) and are connected using conjunctions like dass, weil, wenn, obwohl. The most important difference: in a subordinate clause, the verb goes **to the end of the sentence**!

## 💡 When do we use subordinate clauses?

### 1. dass — that (for facts and statements)

We use **dass** when reporting a fact, information, or opinion.

```
Ich weiß, dass er kommt.
I know that he is coming.
```

```
Er sagt, dass er krank ist.
He says that he is sick.
```

### 2. weil — because (for giving reasons)

We use **weil** to give a reason.

```
Ich lerne Deutsch, weil es Spaß macht.
I learn German because it is fun.
```

```
Er bleibt zu Hause, weil er krank ist.
He stays at home because he is sick.
```

### 3. wenn — if / when (for conditions or repeated events)

We use **wenn** for conditions (if) or when something happens repeatedly.

```
Wenn ich Zeit habe, gehe ich spazieren.
If I have time, I go for a walk.
```

```
Ich freue mich, wenn du kommst.
I am happy when you come.
```

### 4. obwohl — although / even though (for contrast)

We use **obwohl** when the information in the second clause is the opposite of what is expected.

```
Obwohl es regnet, gehe ich spazieren.
Although it is raining, I go for a walk.
```

```
Er kommt, obwohl er krank ist.
He is coming even though he is sick.
```

---

## ⚠️ The Rule: Word Order in Subordinate Clauses

| Main Clause | Conjunction | Subordinate Clause |
|:-----------:|:----------:|:------------------:|
| Ich weiß, | **dass** | er **kommt**. |
| Ich lerne Deutsch, | **weil** | es Spaß **macht**. |
| Ich freue mich, | **wenn** | du **kommst**. |
| Er kommt, | **obwohl** | er krank **ist**. |

### The Golden Rule:
```
In a subordinate clause (Nebensatz) ← the verb goes to the END of the sentence!
```

### Difference between Hauptsatz and Nebensatz:

| Main Clause (Hauptsatz) | Subordinate Clause (Nebensatz) |
|:-----------------------:|:-----------------------------:|
| Verb is in **second position** | Verb is at the **end** of the clause |
| Can stand alone | **Cannot** stand alone |
| **Ich komme morgen.** | **..., weil ich morgen komme.** |

## ✅ Example Sentences with Translation

| Sentence | Translation |
|----------|------------|
| Ich denke, dass er Recht hat. | I think that he is right. |
| Sie sagt, dass sie müde ist. | She says that she is tired. |
| Wir bleiben zu Hause, weil es kalt ist. | We stay at home because it is cold. |
| Er ruft an, weil er Hilfe braucht. | He calls because he needs help. |
| Wenn ich Hunger habe, esse ich etwas. | When I am hungry, I eat something. |
| Wenn du willst, können wir gehen. | If you want, we can go. |
| Obwohl er wenig Zeit hat, hilft er mir. | Even though he has little time, he helps me. |
